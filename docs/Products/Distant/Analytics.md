# Analytics & Insights

Track your focus patterns and progress over time.

## Session History

View all your past focus sessions.

### Accessing History

**From Menu:**
- Tap menu â†’ History
- See all completed sessions
- Chronological order
- Most recent first

### Session Details

**Each Session Shows:**
- Focus type and icon
- Duration (planned + overtime)
- Completion status
- Date and time
- XP earned
- Rating (if provided)
- Notes (if added)

### Filtering History

**By Focus Type:**
- Tap filter icon
- Select focus type
- See only that type's sessions
- Clear to see all

**By Date:**
- Scroll through chronological list
- Jump to specific dates
- Review past periods

### Editing Sessions

**What You Can Edit:**
- Session rating (1-5 stars)
- Session notes
- Trim/restore overtime

**What You Can't Edit:**
- Duration
- Focus type
- Date/time
- XP earned

### Deleting Sessions

**How To:**
1. Swipe left on session
2. Tap "Delete"
3. Confirm deletion
4. Session removed permanently

**Note:** XP is not refunded when deleting sessions.

## Statistics Overview

Key metrics about your focus practice.

### Total Focus Time

**What It Shows:**
- All-time total minutes focused
- Displayed in hours and minutes
- Includes all completed sessions
- Lifetime achievement

### Session Count

**What It Shows:**
- Total number of sessions completed
- All focus types combined
- Milestone tracking
- Progress indicator

### Average Session Duration

**What It Shows:**
- Mean length of your sessions
- Calculated from all sessions
- Indicates typical focus length
- Helps set realistic goals

### Completion Rate

**What It Shows:**
- Percentage of sessions completed vs stopped early
- Measures consistency
- Higher is better
- Goal: 90%+

### Current Streak

**What It Shows:**
- Consecutive days with at least one session
- Resets if you miss a day
- Motivation metric
- Daily practice indicator

### Longest Streak

**What It Shows:**
- Your best streak ever
- Personal record
- Goal to beat
- Achievement milestone

## Using Analytics Effectively

### Daily Check-In

**Morning:**
- Check current streak
- Review yesterday's sessions
- Set today's intention

**Evening:**
- Log today's sessions
- Check daily progress
- Maintain streak

### Weekly Review

**Every Week:**
1. Review 7-day chart
2. Check total focus time
3. Note patterns
4. Adjust strategy

**Questions to Ask:**
- Am I focusing consistently?
- Which days are strongest/weakest?
- Am I using varied focus types?
- Is my completion rate improving?

### Monthly Analysis

**Every Month:**
1. Review total statistics
2. Check longest streak
3. Analyze focus type distribution
4. Set goals for next month

**Insights to Gather:**
- Monthly focus time total
- Most productive days
- Focus type preferences
- Areas for improvement

### Goal Setting

**Based on Data:**
- Set realistic daily targets
- Aim for consistency
- Gradually increase duration
- Balance focus types

**Example Goals:**
- Increase average session to 45 minutes
- Maintain 7-day streak
- 90% completion rate
- 20 hours focus this month
