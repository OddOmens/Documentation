# Customization & Settings

Personalize Depthful to match your preferences.

## Display Options

Customize what you see in the thought list.

### Thought Content

**Show Full Text:**
- Display complete thought content in list
- See everything at a glance
- Good for short thoughts
- More scrolling required

**Show Preview:**
- Display first few lines only
- Cleaner, more compact view
- Better for long thoughts
- Tap to see full content

**Toggle:** Settings → Display → Show Full Text

### Photos Display

**Show Photos in List:**
- Display photo thumbnails in thought cards
- Visual identification
- See images without opening
- More visual list

**Hide Photos in List:**
- Cleaner, text-focused view
- Faster scrolling
- Photos still visible when editing
- Minimalist approach

**Toggle:** Settings → Display → Show Photos

### Photo Count Indicators

**Show Image Count:**
- Badge showing number of photos
- Quick reference
- Know which thoughts have media
- Small visual indicator

**Hide Image Count:**
- Cleaner appearance
- Less visual clutter
- Discover photos when opening

**Toggle:** Settings → Display → Show Image Count

### Recording Count Indicators

**Show Recording Count:**
- Badge showing number of voice recordings
- Quick reference
- Know which thoughts have audio
- Small visual indicator

**Hide Recording Count:**
- Cleaner appearance
- Less visual clutter
- Discover recordings when opening

**Toggle:** Settings → Display → Show Recording Count

### Tags Display

**Show All Tags:**
- Display all tags on thought card
- Complete information
- Easy filtering
- More space used

**Show First Tag Only:**
- Display only first tag
- Cleaner appearance
- Less clutter
- Still organized

**Hide Tags:**
- No tags in list view
- Minimalist approach
- Tags still applied
- See when editing

**Toggle:** Settings → Display → Tag Display

### Timestamps

**Show Timestamps:**
- Display creation date/time
- Track when thoughts were written
- Chronological context
- More information

**Hide Timestamps:**
- Cleaner appearance
- Focus on content
- Less visual noise
- Timestamps still recorded

**Toggle:** Settings → Display → Show Timestamps

## Tips for Customization

### Finding Your Style

1. **Start with defaults** - Use for a week
2. **Adjust gradually** - Change one thing at a time
3. **Test different views** - Try various combinations
4. **Settle on preferences** - What works for you

### Seasonal Changes

- **Summer:** Light mode, colorful icons
- **Winter:** Dark mode, minimal display
- **Adjust as needed** - Match your mood

### Performance

- **Hiding photos** - Faster scrolling
- **Show preview** - Less data to render
- **Minimal display** - Best performance

### Organization

- **Show all tags** - Better organization
- **Show timestamps** - Chronological tracking
- **Show counts** - Media awareness
