# Reports & Assets

Generate financial reports and track net worth.

## Reporting

### Report Types

**Current Year:**
- January 1 to today
- Year-to-date summary

**Previous Year:**
- Full previous calendar year
- Year-end totals

**All Time:**
- Complete transaction history
- Lifetime totals

**Current Month:**
- First of month to today
- Month-to-date

**Previous Month:**
- Full previous month
- Monthly comparison

**Custom Range:**
- Select start date
- Select end date
- Any time period

### Report Contents

**Summary:**
- Net income
- Total revenue
- Total expenses
- Transaction count

**Transaction List:**
- All expenses
- All revenue
- Sorted by date or amount
- Full details

### Report Actions

**Export CSV:**
- Download as spreadsheet
- Import to accounting software
- Share with accountant
- Tax preparation

**Edit Transactions:**
- Swipe to edit
- Update details
- Correct errors

**Delete Transactions:**
- Swipe to delete
- Remove duplicates
- Clean up data

### Use Cases

**Tax Preparation:**
- Generate year-end report
- Export for accountant
- Review deductions
- Verify income

**Monthly Review:**
- Compare to previous month
- Track spending trends
- Adjust budget
- Monitor progress

**Quarterly Analysis:**
- Custom range for quarter
- Business performance
- Trend analysis
- Planning

## Asset Manager

Track all assets to calculate net worth.

### Asset Categories

**Cash & Bank Accounts:**
- Checking accounts
- Savings accounts
- Money market accounts

**Investments:**
- Stocks
- Bonds
- Mutual funds
- ETFs

**Real Estate:**
- Primary residence
- Rental properties
- Land

**Vehicles:**
- Cars
- Motorcycles
- Boats
- RVs

**Personal Property:**
- Jewelry
- Art
- Collectibles
- Electronics

**Retirement Accounts:**
- 401(k)
- IRA
- Roth IRA
- Pension

**Business Interests:**
- Business ownership
- Partnerships
- Equity

**Cryptocurrencies:**
- Bitcoin
- Ethereum
- Other crypto

**Other Assets:**
- Anything else of value

### Adding Assets

1. Go to Asset Manager
2. Tap "+"
3. Enter details:
   - Asset name
   - Category
   - Current value
   - Wallet (optional)
4. Save

### Asset Features

**Value Tracking:**
- Historical value chart
- Daily averages
- Total change
- Percent change

**Organization:**
- Assign to wallets
- Sort by value, name, category
- Filter by wallet
- Search by name

**Performance:**
- Track appreciation/depreciation
- View trends over time
- Compare assets
- Monitor net worth

### Updating Values

1. Tap asset
2. Update current value
3. Save
4. History automatically tracked

### Net Worth Calculation

**Total Assets:**
- Sum of all asset values
- Real-time calculation
- Comprehensive view

**Tips:**
- Update values monthly
- Track both appreciating and depreciating
- Include all assets
- Regular review

## Data Management

### Force Resave

**When to Use:**
- Totals seem incorrect
- Data inconsistencies
- After updates

**What It Does:**
- Recalculates all totals
- Fixes data issues
- Refreshes calculations

### Delete Data

**Options:**
- Wallet Data
- Subscription Data
- Revenue Data
- Expense Data
- Card Data
- Financial Goal Data
- All Data

::: warning Permanent Deletion
All deletions are permanent and cannot be undone. Back up important data before deleting.
:::
