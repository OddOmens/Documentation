# Features

Explore everything Breatheful has to offer.

## Breathing Exercises

### 14 Breathing Modes

Each mode has a unique timing pattern (Inhale-Hold-Exhale-Hold):

- **Quick (2-0-2-0)** - Fast breathing for quick sessions
- **Shallow (3-0-3-0)** - Gentle breathing pattern
- **Standard (4-0-4-0)** - Default balanced breathing
- **Calm (4-4-4-4)** - Box breathing with holds
- **Sleep (4-7-8-0)** - Designed to help you fall asleep
- **Focus (5-0-5-0)** - Longer breaths for concentration
- **Deep Zen (8-4-8-4)** - Extended breathing with holds
- **Monk (10-5-8-5)** - Advanced deep breathing
- **Alternate Nostril (2-5-2-5)** - Short breaths with long holds
- **Calming (7-0-11-0)** - Extended exhale for relaxation
- **Box Breathing (4-4-4-4)** - Military-style breathing
- **Relaxing (4-7-8)** - Similar to sleep mode
- **Energizing (3-0-2-0)** - Quick inhale, faster exhale
- **Ocean Breath (5-2-5-2)** - Balanced with short holds

## Panic Attack Support

Quick access to grounding exercises when you need them most.

### How to Access
- **Swipe left** from the main screen to access panic attack mode
- **Swipe right** to return to main breathing screen

### 5-4-3-2-1 Grounding Exercise

A proven technique to calm anxiety:

1. **Name 5 things you can see** - Look around and identify objects
2. **Find 4 things you can touch** - Feel textures and surfaces
3. **Listen for 3 sounds you can hear** - Notice ambient sounds
4. **Notice 2 things you can smell** - Identify scents around you
5. **Focus on 1 thing you can taste** - Notice taste in your mouth

### Calming Breath Pattern

**Pattern (4-2-6-1)** - Extended exhale to activate parasympathetic nervous system and promote calm.
