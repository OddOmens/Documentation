## Appearance

### Theme

**Light/Dark Mode:**
- Toggle between light and dark themes
- Affects entire app interface
- Choose what's comfortable for your eyes
- Can match system settings

### Accent Colors

**24 Color Options:**
- 3 rows of 8 colors each
- Organized by intensity
- Affects highlights, progress circles, and accents
- Preview before selecting

**Color Palette:**
- Blues, greens, purples
- Reds, oranges, yellows
- Pinks, teals, neutrals
- Find your perfect match

### Checkbox Style

**Square (Default):**
- Traditional checkbox appearance
- Clear and familiar
- Works well in lists

**Circle:**
- Rounded checkbox style
- Modern and soft
- Personal preference

### Habit View Style

**List (Traditional):**
- Full details visible
- Expandable items
- Progress bars shown
- Best for detailed tracking

**Badge (Compact):**
- Week view with 7-day strip
- Shows past 3 and next 3 days
- Space-efficient
- Great for multiple habits
- Swipe through weeks

### App Icon

**Multiple Icon Options:**
- Various designs and colors
- Preview before selecting
- Customize your home screen
- Match your aesthetic

## View Options

### Time-Based Filtering

**What it does:**
- Organizes items by time periods
- Creates sections: Morning, Afternoon, Evening, Today
- Groups items by when they occur
- Easier to focus on current time period

**Sections:**
- **Morning** - Early day items
- **Afternoon** - Midday items
- **Evening** - End of day items
- **Today** - Items without specific times

**Toggle:** Enable/disable in Customize settings

### Hide Completed Items

**What it does:**
- Removes completed habits/tasks from view
- Declutters your daily view
- Focus on what's left to do
- Completed items still tracked

**Best for:**
- Minimalist view
- Focusing on incomplete items
- Reducing visual noise
- End-of-day cleanup

**Toggle:** Enable/disable in Customize settings

**Sleep Time:**
- Set your evening end time
- Customize label (8 options):
  - "Sleep time"
  - "Bedtime"
  - "Dusk"
  - "Evening ends"
  - "End of day"
  - "Last light"
  - "Day ends"
  - "Night time"

**Time Pickers:**
- Easy time selection
- 24-hour or 12-hour format
- Adjust anytime

## Reminders

### Show Reminders Section

**What it does:**
- Toggles date reminders visibility
- Shows/hides reminders in main view
- Hides from main menu when disabled

**When to disable:**
- If you don't use date reminders
- To simplify main view
- To reduce clutter

**When to enable:**
- If you track birthdays, appointments
- For important date-based reminders
- To see reminders in daily view

## Customization Tips

### Finding Your Style

1. **Experiment** - Try different colors and themes
2. **Match your mood** - Change colors seasonally
3. **Consider time of day** - Dark mode for evening
4. **Personal preference** - No wrong choices

### Optimizing Your View

1. **Try both habit styles** - List vs Badge
2. **Use time-based filtering** - If you have many items
3. **Hide completed** - For minimalist view
4. **Adjust bookends** - Match your actual schedule

### Notification Strategy

1. **Start with all enabled** - See what's helpful
2. **Disable what's annoying** - Reduce notification fatigue
3. **Keep important ones** - Habit reminders for new habits
4. **Adjust as habits form** - Disable when automatic

### Seasonal Changes

1. **Update colors** - Match seasons or holidays
2. **Adjust wake/sleep times** - As daylight changes
3. **Change app icon** - Refresh your home screen
4. **Review settings** - Quarterly check-in
