# Habit Tracking

Build better habits with powerful tracking, goals, and streak monitoring.

## Creating Habits

### Via Quick Add
1. Type habit name in Quick Add bar
2. Select "Habit" type
3. Press return
4. Double-tap later to add details

### Via Menu
1. Tap menu (three dots) → Manage Habits → + button
2. Fill in details:
   - **Title** (required) - Name of your habit
   - **Description** (optional) - Additional context
   - **Start date** - When to begin tracking
   - **End date** (optional) - For temporary habits
   - **Time** (optional but recommended) - When to do it
   - **Goal tracking** (optional) - Measure progress
   - **Reminders** (optional) - Get notified

## Habit Types

### Simple Habits

Binary completion tracking - either done or not done.

**Examples:**
- Make bed
- Meditate
- Take vitamins
- Journal
- Exercise

**Features:**
- Checkbox completion
- Streak tracking
- Simple yes/no tracking

### Goal-Based Habits

Track measurable progress toward daily, weekly, monthly, or yearly goals.

**Examples:**
- Read 30 minutes
- Drink 8 glasses of water
- Run 3 miles
- Practice piano 1 hour
- Write 500 words

**Features:**
- Progress tracking with +/- buttons
- Multiple time period goals
- 19 measurement units
- Visual progress bars
- Calendar view with color-coded progress

### Time
- **Minutes** - Reading, meditation, exercise

### Activity
- **Sessions** - Gym sessions, meetings
- **Played** - Musical instruments, games
- **Practiced** - Skills, languages
- **Listened** - Podcasts, audiobooks

### Reading & Writing
- **Pages** - Books, documents
- **Written** - Words, paragraphs
- **Read** - Articles, chapters

### Exercise
- **Miles** - Running, cycling, walking
- **Laps** - Swimming, track
- **Reps** - Strength training
- **Sets** - Workout routines

### Learning
- **Lessons** - Courses, tutorials
- **Completed** - Tasks, modules
- **Solved** - Problems, puzzles

### Performance
- **Acts** - Performances, presentations

### Health
- **Glasses** - Water intake
- **Slept** - Sleep hours

## Habit States

Habits can be in three states:

### 1. Incomplete (Empty Checkbox)
- Habit not yet completed today
- Default state
- Doesn't affect streak

### 2. Completed (Checked Checkbox)
- Habit completed for the day
- Builds your streak
- Counts toward success rate

### 3. Skipped (Ring Checkbox)
- Intentionally skipped (rest day, not applicable)
- **Doesn't break your streak**
- Doesn't count as completion
- Perfect for planned rest days

**To cycle states:** Tap the checkbox repeatedly
```
Incomplete → Complete → Skip → Incomplete
```

::: tip Use Skip for Rest Days
If you have a "Go to gym" habit but take Sundays off, skip it on Sundays to maintain your streak without false completions.
:::

## Goal Progress Tracking

### In List View

1. Tap habit to expand details
2. See progress bar and current/goal values
3. Use **+/-** buttons to adjust progress
4. Choose increment value (1, 5, 10, 20, 50)
5. Progress saves automatically

**Example:**
```
Read 30 minutes
Progress: 15/30 minutes
[- 1 5 10 +]
```

### In Habit Form

When editing a goal-based habit:

1. View calendar with color-coded progress
2. See daily, weekly, monthly, or yearly totals
3. Track progress across different time periods
4. Visualize completion patterns

**Color coding:**
- Light color: Partial progress
- Full color: Goal met or exceeded
- No color: No progress

## Habit View Styles

### List Style (Default)

Traditional list view with full details.

**Features:**
- Checkboxes for completion
- Shows time, description, and progress
- Expandable details
- Progress bars for goals
- +/- buttons when expanded

**Best for:**
- Detailed tracking
- Seeing all information at once
- Traditional list lovers

### Badge Style

Compact badge view with 7-day week strip.

**Features:**
- Shows completion status for past 3 days and next 3 days
- Swipe through weeks
- Compact, space-efficient design
- Quick visual overview
- Great for tracking multiple habits

**Best for:**
- Seeing weekly patterns
- Tracking many habits
- Visual learners
- Quick check-ins

**How to switch:**
Settings → Customize → Habit View Style

## Streak Tracking

### Current Streak
- Number of consecutive days completed
- Shown in habit details
- Resets when you miss a day (unless skipped)

### Longest Streak
- Your all-time best streak
- Shown in analytics
- Personal record to beat

### Streak Rules
- **Completing** a habit extends the streak
- **Missing** a habit breaks the streak
- **Skipping** a habit maintains the streak
- Streaks are per-habit, not global

::: warning Streak Tips
- Set realistic habits you can do daily
- Use reminders to stay consistent
- Skip intentionally rather than forgetting
- Don't let one missed day discourage you
:::

## Habit Management

Access via Menu → Manage Habits

### Features

**View All Habits:**
- Active habits
- Archived habits
- Search and filter
- Sort by name or time

**Edit Mode:**
- Select multiple habits
- Archive/unarchive in bulk
- Delete multiple habits
- Bulk operations

**Individual Actions:**
- Tap to edit
- Swipe to delete
- View details
- Check archive status

### Archiving Habits

**Why archive?**
- Removes from daily view
- Preserves all historical data
- Can be unarchived anytime
- Keeps active list manageable

**How to archive:**
1. Menu → Manage Habits
2. Tap Edit
3. Select habits to archive
4. Tap Archive button

**Archived habits:**
- Archive date recorded
- All data preserved
- Streaks and progress saved
- Can view in analytics

**To unarchive:**
1. Menu → Manage Habits
2. Scroll to Archived section
3. Tap Edit
4. Select habits
5. Tap Unarchive

## Habit Reminders

Set multiple reminder times for each habit.

### Setup

1. Edit habit → Enable reminders
2. Tap "Add Reminder"
3. Set time
4. Enable/disable toggle
5. Add multiple reminders if needed

### Features

- **Multiple reminders per habit**
- **Individual enable/disable toggles**
- **Swipe to delete**
- **Respects Do Not Disturb**
- **Customizable times**

### Best Practices

- Set reminders slightly before habit time
- Use multiple reminders for important habits
- Disable reminders once habit is automatic
- Adjust times based on your schedule

## Tips for Successful Habit Tracking

### Starting Out

1. **Start with 3-5 habits** - Don't overwhelm yourself
2. **Choose specific times** - "Exercise at 7 AM" vs "Exercise sometime"
3. **Stack habits** - Attach new habits to existing routines
4. **Set reminders** - Until habits become automatic

### Building Consistency

1. **Track daily** - Check off habits as you complete them
2. **Review weekly** - Check analytics for patterns
3. **Adjust goals** - Lower if consistently missing, raise if too easy
4. **Celebrate streaks** - Acknowledge your progress

### Using Goals Effectively

1. **Start small** - 10 minutes is better than 0 minutes
2. **Be specific** - "Read 20 pages" vs "Read more"
3. **Track honestly** - Accurate data helps you improve
4. **Review progress** - Use calendar view to see patterns

### Maintaining Motivation

1. **Check achievements** - See what you've unlocked
2. **View analytics** - Watch your success rate grow
3. **Compare periods** - See improvement over time
4. **Share progress** - Tell friends about your streaks

### When You Miss a Day

1. **Don't give up** - One missed day doesn't erase progress
2. **Analyze why** - Was the goal too ambitious?
3. **Adjust if needed** - Make habits more realistic
4. **Start again** - Begin a new streak immediately

### Advanced Strategies

1. **Use skip strategically** - Plan rest days in advance
2. **Batch similar habits** - Group by time or type
3. **Create habit chains** - One habit triggers the next
4. **Track keystone habits** - Habits that trigger other positive behaviors
