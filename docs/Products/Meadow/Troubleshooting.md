# Troubleshooting

Solutions to common issues.

## Data & Sync Issues

### Data isn't syncing between devices

**Solutions:**
1. Check iCloud status in **Myself** tab
2. Verify signed into same iCloud account
3. Check iCloud Drive is enabled
4. Ensure Meadow has iCloud permission
5. Check internet connection
6. Wait a few minutes for sync

### See "Local Storage Only" but want iCloud

**Solutions:**
1. Sign into iCloud: Settings → Sign in
2. Enable iCloud Drive
3. Enable Meadow in iCloud settings
4. Force quit and reopen Meadow

### Data disappeared

**Solutions:**
1. If using iCloud: Sign back into same account
2. Force quit and reopen app
3. Wait for sync to complete
4. Check other devices

### Sync stuck on "Syncing..."

**Solutions:**
1. Check internet connection
2. Wait 5-10 minutes
3. Force quit and reopen
4. Check Apple System Status
5. Try again later

## App Performance Issues

### App is slow or laggy

**Solutions:**
1. Check device storage
2. Close background apps
3. Restart iPhone
4. Update iOS
5. Update Meadow

### App crashes on launch

**Solutions:**
1. Force quit completely
2. Restart iPhone
3. Check for updates
4. Ensure 1GB free storage
5. Contact support if persists

### Keyboard doesn't dismiss

**Solutions:**
1. Tap "Done" button
2. Tap outside text field
3. Swipe down on keyboard
4. Force quit and reopen

## Form & Input Issues

### Can't save item

**Required fields:**
- **Conversations**: Title and Support Person
- **Goals**: Category, Goal, and Description
- **Thoughts**: Content
- **Supporters**: First and Last Name

**Solutions:**
1. Fill all required fields
2. Ensure fields aren't just spaces
3. Check save button (white = valid, gray = invalid)

### Support person doesn't appear

**Solutions:**
1. Go to **Supporters** tab
2. Verify person exists
3. Add them if missing
4. Return to **Chats** and try again

### Can't add profile photo

**Solutions:**
1. Grant photo access in Settings
2. Try taking new photo
3. Restart app
4. Check photos in library

## Navigation Issues

### Bottom navigation disappeared

**Solutions:**
1. Dismiss keyboard
2. Scroll to top
3. Force quit and reopen

**Note:** Navigation bar hides when keyboard is visible (intentional).

### Can't go back

**Solutions:**
1. Look for back arrow
2. Save or cancel current form
3. Tap outside modal sheets
4. Force quit and reopen

## Settings Issues

### App icon won't change

**Solutions:**
1. Ensure iOS 17.0+
2. Try different icon
3. Force quit and check home screen
4. Restart iPhone

### Navigation labels won't toggle

**Solutions:**
1. Toggle off and on again
2. Force quit and reopen
3. Check setting persists

## Deletion Issues

### Can't delete item

**Solutions:**
1. Tap item to open
2. Look for trash icon
3. Tap and confirm
4. For bulk: Settings → Select Data to Delete

### Accidentally deleted something

**Unfortunately, deletions are permanent and cannot be undone.**

**Prevention:**
- Read deletion alerts carefully
- Keep iCloud sync enabled
- Be cautious with bulk deletion

## Getting Help

### Before Contacting Support

1. Check this guide
2. Try basic troubleshooting:
   - Force quit app
   - Restart iPhone
   - Check for updates
   - Verify internet

### Contact Support

**Email:** support@oddomens.com

**Include:**
- Device model
- iOS version
- Meadow version
- Detailed description
- Steps to reproduce
- Screenshots if applicable

### In-App Support

1. Go to **Myself** tab
2. Tap Settings icon
3. Scroll to "Help" section
4. Choose:
   - Email Support
   - Help Center
   - Report an Issue
   - Request a Feature

## Preventing Issues

### Best Practices

1. Keep iCloud sync enabled
2. Update regularly
3. Maintain storage (1GB+ free)
4. Be careful with deletions
5. Force quit occasionally
6. Restart device monthly

### Data Safety

1. Enable iCloud sync
2. Don't delete impulsively
3. Keep device backed up
4. Use multiple devices if possible

## Known Limitations

Current limitations (not bugs):
- No data export (planned)
- No undo for deletions
- No search functionality (planned)
- No filtering options (planned)
- No reminders (use iOS Calendar)
- No widgets (planned)
- No Apple Watch app (planned)

## Still Having Issues?

1. Email support@oddomens.com with details
2. Check for updates in App Store
3. Reinstall app (last resort - may lose local data)

**Remember:** Most issues resolve with force quit and restart!
