# Import & Export

Backup your thoughts and move data between devices.

## Export Formats

Depthful supports multiple export formats for flexibility.

### Text File (.txt)

**What It Includes:**
- All thought content
- Organized by tags
- Creation dates
- Simple, readable format

**Best For:**
- Reading on any device
- Simple backups
- Sharing with others
- Universal compatibility

**Format Example:**
```
=== Gratitude ===

November 12, 2025
Beautiful sunrise this morning...

November 11, 2025
Grateful for good health...

=== Ideas ===

November 10, 2025
App idea: Meal planner...
```

### Markdown (.md)

**What It Includes:**
- Formatted text with markdown
- Headers and lists
- Tags as headers
- Timestamps
- Preserves formatting

**Best For:**
- Formatted documents
- Publishing to blogs
- Markdown editors
- Preserving structure

**Format Example:**
```markdown
# Gratitude

## November 12, 2025
**Beautiful sunrise** this morning...

## November 11, 2025
Grateful for *good health*...

# Ideas

## November 10, 2025
App idea: Meal planner
- Weekly planning
- Shopping lists
```

### CSV (.csv)

**What It Includes:**
- All thought attributes
- Content, tags, timestamps
- Spreadsheet-compatible
- Complete metadata

**Best For:**
- Data analysis
- Spreadsheet import
- Database import
- Complete backups

**Columns:**
- ID
- Content
- Tags
- Created Date
- Updated Date
- Is Favorite

### Custom Tags Export

**What It Includes:**
- Custom tag names
- Tag colors
- Tag definitions only

**Best For:**
- Moving tags to another device
- Backing up tag system
- Sharing tag schemes
- Restoring tags

## Exporting Data

### Export All Thoughts

**How To:**
1. Open menu
2. Tap "Export"
3. Choose format (Text, Markdown, or CSV)
4. Select destination
5. Share or save

**Options:**
- Export to Files app
- Share via Mail
- Send via Messages
- Save to cloud storage

### Export Single Thought

**How To:**
1. Open the thought
2. Tap share icon
3. Choose format
4. Select destination
5. Share or save

**Use Cases:**
- Share specific thought
- Backup important entry
- Send to someone
- Save separately

### Export Custom Tags

**How To:**
1. Open menu
2. Tap "Export Custom Tags"
3. Save JSON file
4. Store safely

**Why Export Tags:**
- Backup tag definitions
- Move to new device
- Share with others
- Restore after reinstall

## Importing Data

### Import Thoughts

**Supported Formats:**
- Text files (.txt)
- Markdown files (.md)
- CSV files (.csv)

**How To:**
1. Open menu
2. Tap "Import"
3. Select file
4. Choose format
5. Confirm import

**What Happens:**
- Thoughts are added to your journal
- Timestamps preserved (if in CSV)
- Tags applied (if in CSV)
- Existing thoughts not affected

### Import Custom Tags

**How To:**
1. Open menu
2. Tap "Import Custom Tags"
3. Select JSON file
4. Confirm import

**What Happens:**
- Custom tags added
- Colors preserved
- Existing tags not affected
- Ready to use immediately

### Import Compatibility

**From Depthful:**
- Full compatibility
- All data preserved
- Perfect for device transfers

**From Other Apps:**
- Text and Markdown work
- CSV needs proper format
- May need formatting
- Test with small file first

## Moving Between Devices

### New Device Setup

**Steps:**
1. Install Depthful on new device
2. Export from old device
3. Transfer files (AirDrop, cloud, etc.)
4. Import on new device
5. Verify all data

**What Transfers:**
- All thoughts
- Custom tags
- Timestamps
- Favorites
- Tags

**What Doesn't Transfer:**
- Voice recordings (export separately)
- Photos (export separately)
- App settings
- Display preferences

### Keeping Devices in Sync

**Manual Sync:**
1. Export from primary device
2. Import to secondary device
3. Repeat regularly
4. Merge if needed

::: warning No Automatic Sync
Depthful doesn't have automatic cloud sync. All data stays local for privacy. Use manual export/import to sync devices.
:::

## Data Management

### Archiving Old Thoughts

**Why Archive:**
- Reduce app data size
- Organize by time period
- Keep app responsive
- Preserve history

**How To:**
1. Export thoughts from specific period
2. Save with descriptive name
3. Delete from app (optional)
4. Store archive safely

**Example:**
- "Depthful_2024.csv"
- "Thoughts_Q1_2025.md"
- "Journal_January.txt"

### Selective Export

**By Tag:**
1. Filter by tag
2. Export visible thoughts
3. Save by category
4. Organize archives

**By Date:**
1. Sort by date
2. Export specific period
3. Archive by timeframe
4. Clean organization

## Troubleshooting

### Export Issues

**Export Fails:**
- Check storage space
- Try smaller export
- Different format
- Restart app

**File Won't Open:**
- Check file extension
- Try different app
- Re-export
- Verify format

### Import Issues

**Import Fails:**
- Check file format
- Verify file isn't corrupted
- Try smaller file
- Check format compatibility

**Missing Data:**
- CSV format preserves most data
- Text/Markdown lose some metadata
- Voice/photos don't export
- Re-import if needed

### File Management

**Can't Find Export:**
- Check Files app
- Look in recent files
- Check share destination
- Re-export if needed

**File Too Large:**
- Export by date range
- Export by tag
- Multiple smaller files
- Compress if needed

## Best Practices

### Regular Exports

1. **Weekly backups** - Export all thoughts
2. **Monthly archives** - Save by month
3. **Yearly compilations** - Annual review
4. **Before updates** - Safety backup

### Organization

1. **Descriptive names** - Include date and type
2. **Consistent location** - Same folder always
3. **Version control** - Keep multiple versions
4. **Clean up old** - Delete outdated backups

### Security

1. **Encrypt sensitive exports** - Use secure storage
2. **Password protect** - If sharing
3. **Secure cloud** - Use trusted services
4. **Local copies** - Don't rely only on cloud
