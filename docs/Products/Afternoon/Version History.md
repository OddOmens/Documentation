# Afternoon Version History

::: warning
This version history log is maintained by the product development team. For questions about specific versions or features, please view the feature and issue page or contact the support team.
:::

### **Release 2025.09.1**

- **Release Date:** September 2025
- **Release Notes:**
    - Added support for iOS26 and Liquid Glass
    - Added options to set multiple reminders for habits beyond the schedule time
    - Added Option to Skip Habit (Tap Checkbox twice)
    - Added "Time-Based filtering" Enabling this will put habits and tasks into Morning, Afternoon, Evening and Today filters.
    - Added "Hide Completed Items" Enabling this will hide any completed task or habit. To see, disable the feature.
    - Added Weekly Badge View (Set in Customization)
    - Added Support Popup every 10 launches
    - Added Documentation link in Settings under Help in Settings
    - Updated position of notifications
    - Updated position of the dismiss button for notifications
    - Updated the review popup to show on the 2nd and 5th launch
    - Fixed quick add habit, tasks and reminder date bug
    - Fixed Analytics to correctly show years data
    - Fixed notification reminder bug
