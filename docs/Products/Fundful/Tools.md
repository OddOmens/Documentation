# Financial Tools

Powerful calculators and tools for financial planning.

## Breakdown Calculator

Split revenue into percentages for different purposes.

### How to Use

1. Enter total revenue amount
2. Add breakdown items
3. Set percentage for each
4. View calculated dollar amounts
5. Save as template (optional)

### Pre-built Templates

**Business Revenue:**
- Operating Costs, Taxes, Profit, Growth, Emergency Fund

**Freelancer Revenue:**
- Take-home, Taxes, Business Expenses, Retirement, Development

**50/30/20 Budget:**
- Needs (50%), Wants (30%), Savings (20%)

**Savings Priority:**
- Emergency Fund, Retirement, Goals

**Debt Payoff:**
- Minimum Payments, Extra Payments, Living Expenses

**Investment Portfolio:**
- Stocks, Bonds, Real Estate, Cash

### Tips

- Total must equal 100%
- Create templates for recurring income
- Adjust percentages based on goals
- Use for tax planning

## Loan Calculator

Calculate monthly payments and total interest.

### Inputs

- Loan amount
- Down payment
- Interest rate (%)
- Term (months)

### Outputs

- Monthly payment
- Principal amount
- Total interest paid
- Total cost of loan

### Use Cases

- Mortgages
- Car loans
- Personal loans
- Business loans
- Student loans

### Tips

- Compare different terms
- See impact of down payment
- Calculate before committing
- Factor in total interest cost

## Merchant Summary

Analyze spending patterns by merchant.

### Features

- Total spending per merchant
- Transaction count
- Last transaction date
- Filter by year or all years
- Sort by spending amount

### How to Use

1. Open Merchant Summary
2. Select year or "All Years"
3. Review top merchants
4. Swipe between years
5. Identify spending patterns

### Use Cases

- Find forgotten subscriptions
- Negotiate with frequent vendors
- Compare year-over-year spending
- Identify cost-cutting opportunities

## Receipt Scanner

Digitize paper receipts with OCR technology.

### How It Works

1. Select wallet
2. Scan receipt or upload photo
3. OCR extracts data:
   - Merchant name
   - Items
   - Prices
4. Review and edit
5. Select items to save
6. Save as expenses

### Features

- Camera scanning
- Photo upload
- Multi-page support
- Edit before saving
- Auto-saves to Document Locker

### Tips

- Good lighting improves accuracy
- Scan immediately to avoid loss
- Review extracted data carefully
- Keep originals for important receipts

## Document Locker

Secure storage for financial documents.

### Features

- Automatic receipt storage
- Manual document upload
- Organized by month/year
- Grid view with thumbnails
- Full-screen viewing
- Rename documents
- Share documents
- Bulk deletion

### Use Cases

- Store scanned receipts
- Keep tax documents
- Save warranties
- Archive contracts
- Organize invoices

### Tips

- All scanned receipts auto-save here
- Use descriptive names
- Organize by purpose
- Share directly from app
- Regular cleanup
