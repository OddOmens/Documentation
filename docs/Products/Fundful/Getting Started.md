# Getting Started

Start tracking your finances with Fundful.

## First Steps

### 1. Create Your First Wallet

**Choose wallet type:**
- **Personal Wallet** - For personal finances
- **Business Wallet** - For business finances
- **Category Wallet** - For specific spending categories

**How to create:**
1. Tap "+" to add wallet
2. Enter wallet name
3. Select wallet type
4. For business: Choose structure (Sole Proprietorship, LLC, etc.)
5. For category: Select category (Utilities, Groceries, etc.)
6. Tap "Save"

### 2. Add Payment Cards

**Why add cards:**
- Faster expense entry
- Track spending by payment method
- Organize transactions

**How to add:**
1. Go to Payment Cards
2. Tap "+"
3. Enter card details:
   - Card name
   - Cardholder name
   - Last 4 digits
   - Expiration date
   - Card type
4. Tap "Save"

### 3. Set Up Subscriptions

**Track recurring expenses:**
1. Go to Subscriptions
2. Tap "+"
3. Enter details:
   - Merchant name
   - Subscription name
   - Price
   - Renewal date
   - Type (Monthly/Yearly)
4. Tap "Save"

**Auto-expense generation:**
- Active subscriptions automatically create expenses on renewal dates
- Receive notifications before renewals
- Track total monthly and yearly costs

### 4. Start Tracking

**Log your first expense:**
1. Select a wallet
2. Tap "Add Expense"
3. Enter details or scan receipt
4. Save

**Record your first income:**
1. Select a wallet
2. Tap "Add Revenue"
3. Enter amount and source
4. Save

## Understanding the Interface

### Main Screen

**Wallet List:**
- View all wallets
- See year-to-date income
- Check net income
- Tap to open wallet details

**Quick Actions:**
- Add new wallet
- Access tools
- View reports
- Manage settings

### Wallet Detail View

**Summary:**
- Year-to-date income
- Lifetime income
- Net income (revenue - expenses)

**Tabs:**
- Expenses: All expense transactions
- Revenue: All income transactions
- Subscriptions: Recurring subscriptions

**Actions:**
- Add expense
- Add revenue
- Add subscription
- Edit wallet
- Delete wallet

## Quick Start Workflows

### Personal Finance Tracking

1. Create "Personal" wallet
2. Add your payment cards
3. Set up subscriptions (Netflix, Spotify, etc.)
4. Log daily expenses
5. Record income when received
6. Review monthly reports

### Business Finance Tracking

1. Create "Business" wallet with structure
2. Add business payment methods
3. Track business expenses by category
4. Record revenue with hourly or flat rates
5. Use receipt scanner for documentation
6. Export reports for taxes

### Category Budgeting

1. Create category wallets (Groceries, Entertainment, etc.)
2. Log expenses to specific categories
3. Monitor spending per category
4. Set financial goals for each category
5. Review merchant summaries

## Tips for Success

### Daily Habits

- Log expenses immediately
- Scan receipts right away
- Check subscription reminders
- Update revenue when received

### Weekly Review

- Review week's expenses
- Check financial goal progress
- Verify all transactions logged
- Update asset values

### Monthly Tasks

- Generate monthly report
- Review merchant summary
- Update subscription status
- Check goal milestones
- Export for accounting
