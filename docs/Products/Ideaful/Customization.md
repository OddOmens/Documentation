# Customization & Export

Personalize Ideaful and manage your data.

## Themes

**Settings → Theme:**
- Multiple color themes
- App-wide color scheme
- Applies to all UI elements
- Persists across launches

## App Icons

**Settings → App Icon:**
- Multiple icon designs
- Change without leaving app
- Updates immediately
- Seasonal and themed options

## Language

**Settings → Language:**
- Multiple language support
- Full app localization
- Changes all text and labels
- Restart may be required

## Feature Toggles

### Tasks

**Settings → Features → Tasks:**
- **Enable Tasks** - Show/hide task system
- **Show Task Count (Ideas)** - Badge on idea task button
- **Show Task Count (Dashboard)** - Badge on all tasks button
- **Show Task Overview** - Today/Tomorrow cards
- **Show Task Priorities** - Priority filter cards

### Notes

**Settings → Features → Notes:**
- **Show Dashboard Description** - Display descriptions on Dashboard
- **Compact View** - Denser layout with smaller text

### Ideas

**Settings → Features → Ideas:**
- **Show Cancelled Ideas** - Display cancelled ideas
- **Show Completed Ideas** - Display completed ideas
- **Show Archived Ideas** - Display archived ideas

### Notifications

**Settings → Features → Notifications:**
- **Show Device Reminders** - Task reminder notifications
- **Show Device Due Dates** - Due date notifications

### Confirmations

**Settings → Features → Confirmations:**
- **Show Delete Confirmation** - Prompt before deleting ideas

## Data Export

### Export Formats

**Text File (.txt):**
- Plain text format
- Grouped by status
- Includes title, description, notes
- Universal compatibility

**CSV File (.csv):**
- Spreadsheet compatible
- Columns: Status, Title, Description, Notes, Update Date
- Import into Excel, Numbers, Google Sheets
- Data analysis

**Markdown File (.md):**
- Formatted text with headers
- Compatible with note apps
- Preserves structure and hierarchy
- Developer-friendly

### Export Process

1. Settings → Export Ideas
2. Choose format (Text/CSV/Markdown)
3. Tap export button
4. Share via system share sheet
5. Save to Files, email, or other apps

### Use Cases

**Backup:**
- Regular exports for safety
- Multiple format backups
- Cloud storage

**Sharing:**
- Share with team
- Collaborate on ideas
- Present to stakeholders

**Migration:**
- Move to other apps
- Import to project management tools
- Archive old ideas

## Data Management

### iCloud Sync

**Automatic Features:**
- Real-time synchronization
- Across all devices
- Conflict resolution
- Requires iCloud account

**Setup:**
- Enabled by default
- iOS Settings → iCloud
- Ensure iCloud Drive enabled
- Sign in with Apple ID

### Mass Delete

**Settings → Mass Delete Ideas:**
- Deletes ALL ideas permanently
- Confirmation required
- Cannot be undone
- Use with extreme caution

**When to Use:**
- Starting fresh
- Clearing test data
- Major reorganization

### Storage

**Data Storage:**
- CoreData database
- Images in app documents
- Automatic migration
- Efficient management

**Performance:**
- App automatically compresses images
- Regular cleanup recommended
- Delete unused tags
- Archive old ideas
