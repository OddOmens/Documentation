# Effects & Textures

Add depth and character with textures and frame controls.

## Texture Overlays

### Available Textures

- Paper: Classic paper texture
- Canvas: Fabric canvas texture
- Grain: Film grain effect
- Fabric: Woven fabric texture
- And more...

### Texture Controls

**Intensity (0-100%):**
- Strength of texture pattern
- 100%: Full visibility
- 50%: Subtle
- 25%: Very subtle

**Opacity (0-100%):**
- Overall transparency
- 100%: Fully opaque
- 50%: Semi-transparent

**Scale (50-200%):**
- Size of texture pattern
- 100%: Original size
- 50%: Tighter pattern
- 200%: More spread out

**Blend Mode:**
- Multiply: Darkens, adds depth
- Overlay: Balanced blend (most versatile)
- Screen: Lightens, creates glow
- Soft Light: Subtle, gentle
- Hard Light: Strong, dramatic
- Color Dodge: Brightens highlights
- Color Burn: Darkens shadows

**Invert:**
- Flips texture pattern
- Light becomes dark, dark becomes light

### Random Noise

- Click dice icon for instant random texture
- Randomly selects type and settings
- Great for quick styling

### Texture Tips

- Start with 30-50% opacity for subtle effects
- Use multiply for vintage looks
- Use overlay for balanced texture
- Match scale to design size
- Combine with background effects

## Frame & Shadow Controls

### Corner Radius

**Range:** 0-100px
- 0px: Sharp corners (screenshots, technical)
- 10-25px: Subtle rounding (modern, professional)
- 50-100px: Heavy rounding (friendly, approachable)

### Padding

**Range:** 0-200px
- 0px: No padding, edge-to-edge
- 50-100px: Comfortable spacing (recommended)
- 150-200px: Generous spacing, focus on image

### Shadow Customization

**Shadow Opacity (0-100%):**
- 0%: No shadow
- 15-30%: Subtle, professional (recommended)
- 50-100%: Strong, dramatic

**Shadow Blur (0-50px):**
- 0px: Hard shadow edge
- 5-15px: Soft, natural (recommended)
- 30-50px: Very soft, diffused

**Shadow Offset:**
- X Offset (-50 to +50px): Horizontal position
- Y Offset (-50 to +50px): Vertical position
- 0: Centered

**Shadow Color:**
- Default: Black
- Use color picker for custom colors
- Colored shadows create unique effects

**Shadow Position Grid:**
- Click any of 9 positions for instant placement
- Center: No offset (glow effect)
- Corners: Diagonal shadows
- Sides: Directional shadows
- Bottom-right is most natural

### Quick Reset Buttons

- Each control has reset button
- Returns to default value
- Saves time when experimenting

## Style Presets

### Professional Look
- 25px radius
- 75px padding
- 15% opacity
- 8px blur
- 0/4px offset

### Dramatic Look
- 0px radius
- 50px padding
- 30% opacity
- 20px blur
- 0/10px offset

### Minimal Look
- 15px radius
- 100px padding
- 10% opacity
- 5px blur
- 0/2px offset

### Floating Effect
- High blur (30px+)
- Low opacity (10-15%)
- Centered or slight offset
