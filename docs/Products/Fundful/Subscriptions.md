# Subscriptions & Goals

Track recurring subscriptions and achieve financial goals.

## Subscription Management

### Adding Subscriptions

1. Go to Subscriptions
2. Tap "+"
3. Enter details:
   - Merchant name
   - Subscription name
   - Price
   - Renewal date
   - Type (Monthly/Yearly)
4. Save

### How It Works

**Automatic Expense Generation:**
- Active subscriptions create expenses on renewal dates
- Expenses added to associated wallet
- No manual entry needed

**Notifications:**
- Reminders before renewal dates
- Catch unwanted renewals
- Stay informed

### Subscription Status

**Active:**
- Currently subscribed
- Auto-generates expenses
- Counts toward totals

**Inactive:**
- Cancelled subscription
- No expense generation
- Preserves history

### Viewing Subscriptions

**Summary:**
- Total monthly cost
- Total yearly cost
- Number of active subscriptions

**List View:**
- All subscriptions
- Renewal dates
- Prices
- Status

### Managing Subscriptions

**Edit:**
- Update price
- Change renewal date
- Modify details

**Toggle Status:**
- Set to inactive when cancelled
- Reactivate if resubscribed

**Delete:**
- Remove permanently
- Loses history

### Tips

- Review total monthly cost regularly
- Set to inactive instead of deleting
- Enable notifications
- Track free trials
- Identify savings opportunities

## Financial Goals

### Creating Goals

1. Go to Financial Goals
2. Tap "+"
3. Enter details:
   - Goal name
   - Description ("I want to...")
   - Target amount
   - Current amount
   - Due date (optional)
4. Save

### Goal Features

**Progress Tracking:**
- Visual progress bar
- Percentage complete
- Amount remaining
- Current vs target

**Due Dates:**
- Optional deadline
- Automatic reminders
- Notifications:
  - 7 days before
  - 3 days before
  - On due date

### Updating Progress

1. Tap goal
2. Update current amount
3. Save
4. Progress bar updates

### Goal Examples

**Emergency Fund:**
- Target: $10,000
- Description: "Build 6-month emergency fund"
- Track monthly contributions

**Vacation Fund:**
- Target: $5,000
- Description: "Save for Europe trip"
- Due date: Next summer

**Debt Payoff:**
- Target: $15,000
- Description: "Pay off credit card"
- Track payments

**Down Payment:**
- Target: $50,000
- Description: "Save for house down payment"
- Long-term goal

### Tips

- Break large goals into milestones
- Update regularly for motivation
- Enable due dates for urgency
- Use descriptive names
- Celebrate achievements

## Combining Subscriptions & Goals

### Subscription Savings Goal

1. Review subscription costs
2. Cancel unused subscriptions
3. Create goal with savings
4. Track monthly progress

### Example Workflow

1. Total monthly subscriptions: $200
2. Cancel $50 in unused subscriptions
3. Create goal: "Save $600 from subscriptions"
4. Track $50/month savings
5. Reach goal in 12 months
