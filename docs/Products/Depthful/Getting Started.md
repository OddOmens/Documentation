# Getting Started

Start capturing your thoughts with Depthful.

## Installation

1. **Download Depthful** from the App Store
2. **Open the app** - no account or sign-up required
3. **Start writing** immediately

## Creating Your First Thought

1. Tap the **"+" button** at the bottom
2. Start typing your thought
3. Thoughts are **automatically saved** as you type
4. Tap back or swipe down to return to your list

::: tip Auto-Save
Depthful saves your thoughts automatically as you write. No need to worry about losing your work!
:::

## Understanding the Interface

### Main Screen

**Top Bar:**
- **Search icon** - Find thoughts by content
- **Filter icon** - Filter by tags
- **Sort icon** - Change sort order
- **Menu** - Settings and options

**Thought List:**
- All your thoughts displayed as cards
- Tap any thought to open and edit
- Swipe left to delete (with confirmation)
- Favorites appear at the top with star icon

**Bottom Bar:**
- **"+" button** - Create new thought
- **Analytics icon** - View statistics
- **Tags icon** - Manage tags

### Thought Editor

**Toolbar:**
- **Back arrow** - Return to list
- **Star icon** - Mark as favorite
- **Formatting buttons** - Bold, italic, underline, etc.
- **Attach icons** - Add photos or voice recordings

**Content Area:**
- Write freely with full markdown support
- See timestamps (created and last updated)
- View attached photos and recordings

## Basic Operations

### Writing a Thought

1. Tap "+" to create
2. Type your thought
3. Use formatting toolbar for rich text
4. Tap back when done

### Editing a Thought

1. Tap the thought card
2. Make your changes
3. Changes save automatically
4. Tap back to return

### Deleting a Thought

1. Swipe left on the thought card
2. Tap "Delete"
3. Confirm deletion
4. Thought is permanently removed

::: warning Empty Thoughts
Empty thoughts are automatically discarded to keep your journal clean. Add at least some text to save a thought.
:::

## Adding Tags

Tags help you organize and find related thoughts.

### Using Default Tags

1. Open a thought
2. Tap the tag icon or "Add Tags" button
3. Select from 18 default tags:
   - Advice, Daydream, Dream, Experience
   - Goal, Gratitude, Idea, Inspiration
   - Journal, Learning, Memory, Milestone
   - Question, Quote, Reflection, Regret
   - Story, Thought
4. Tap tags to select/deselect
5. Apply multiple tags to one thought

### Creating Custom Tags

1. Tap the tags icon in bottom bar
2. Tap "Create Custom Tag"
3. Enter tag name
4. Choose a color
5. Tap "Save"
6. Use your custom tag on any thought

## Organizing Your Thoughts

### Sorting Options

Tap the sort icon to choose:
- **Newest First** - Most recent at top (default)
- **Oldest First** - Earliest at top
- **Recently Viewed** - Last opened first
- **Least Recently Viewed** - Oldest viewed first
- **A to Z** - Alphabetical by content
- **Z to A** - Reverse alphabetical

::: tip Favorites Always First
Starred thoughts always appear at the top regardless of sort order.
:::

### Filtering by Tags

1. Tap the filter icon
2. Select one or more tags
3. Only thoughts with those tags appear
4. Clear filters to see all thoughts

### Searching

1. Tap the search icon
2. Type keywords
3. Results update as you type
4. Search looks through all thought content

## Adding Media

### Voice Recordings

1. Open a thought
2. Tap the microphone icon
3. Tap record button to start
4. Speak (up to 5 minutes)
5. Tap stop when done
6. Recording is automatically transcribed

**Permissions needed:**
- Microphone access
- Speech Recognition (for transcription)

### Photos

1. Open a thought
2. Tap the photo icon
3. Choose from library or take new photo
4. Photo is automatically compressed and attached
5. Add multiple photos to one thought

**Permission needed:**
- Photos library access

## Tips for Beginners

### Start Simple

- Don't worry about perfect formatting
- Write freely and naturally
- Add tags later if you forget
- Use favorites for important thoughts

### Build a Habit

- Write at the same time each day
- Start with just one thought per day
- Check your streak in Analytics
- Celebrate consistency

### Explore Features Gradually

1. **Week 1:** Just write thoughts
2. **Week 2:** Add tags to organize
3. **Week 3:** Try voice recordings
4. **Week 4:** Attach photos and use formatting

### Keep It Personal

- There's no right way to journal
- Use tags that make sense to you
- Create custom tags for your needs
- Make Depthful work for you
