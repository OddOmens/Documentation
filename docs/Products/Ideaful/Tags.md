# Tags & Organization

Organize ideas with color-coded tags and smart filtering.

## Creating Tags

### Method 1: Quick Create
1. Tap tag icon in Dashboard toolbar
2. Tap **+** in tag filter bar
3. Enter tag name
4. Choose color
5. Save

### Method 2: Settings
1. Settings → Tags
2. Tap **+**
3. Enter tag name
4. Choose color from 12 options
5. Save

## Tag Colors

**12 Available Colors:**
- **Blue** (#4B7BEC) - Projects, work
- **Green** (#26DE81, #20BF6B) - Active, growth
- **Red** (#FC5C65, #FF6B6B) - Urgent, important
- **Orange** (#FD9644) - In progress
- **Purple** (#A55EEA, #5F27CD) - Creative, ideas
- **Teal** (#2BCBBA, #00D2D3) - Personal, wellness
- **Pink** (#E456F0) - Fun, social
- **Light Blue** (#54A0FF) - Planning, research

## Using Tags

### Adding Tags to Ideas

1. Open idea details
2. Tap tags section
3. Select tags to add
4. Tap to toggle on/off
5. Multiple tags allowed

### Tag Display

- Colored pills with tag name
- Shows on Dashboard
- Shows in idea details
- Visual organization

## Tag Filtering

### Single Tag Filter

1. Tap tag icon in Dashboard
2. Tag filter bar appears
3. Tap tag to filter
4. Only tagged ideas show
5. Tap **X** to clear

### Multiple Tag Filter

1. Open tag filter bar
2. Tap multiple tags
3. Ideas with ANY selected tag show
4. Combine for powerful filtering
5. Clear all with **X**

## Bulk Tagging

Apply tags to multiple ideas at once.

### How to Bulk Tag

1. Tap menu (☰) → Bulk Tag Ideas
2. Select multiple ideas by tapping
3. Tap tag icon in toolbar
4. Choose tags to apply
5. Tags added to all selected ideas

### Use Cases

- Tag project-related ideas
- Organize by category
- Prepare for filtering
- Batch organization

## Managing Tags

### Viewing Tags

**Settings → Tags:**
- View all tags
- See tag usage count
- Edit or delete
- Create new tags

### Editing Tags

1. Settings → Tags
2. Tap tag to edit
3. Change name or color
4. Save changes
5. Updates all tagged ideas

### Deleting Tags

1. Settings → Tags
2. Swipe left on tag
3. Confirm deletion
4. Removes from all ideas

## Organization Tips

### Tag Naming

- Keep names short
- Use consistent naming
- Avoid duplicates
- Descriptive but concise

### Color Coding

- Consistent colors for categories
- Red for urgent
- Green for active
- Blue for planning

### Tag Hierarchy

- General to specific
- Category → Subcategory
- Project → Feature

### Regular Maintenance

- Review unused tags
- Delete obsolete tags
- Consolidate similar tags
- Update as needs change

## Advanced Filtering

### Combining Filters

- Tag filter + Status sections
- Multiple tags + Task filters
- Create custom views
- Focus on specific work

### Saved Filters

- Use tag combinations regularly
- Quick access to filtered views
- Consistent organization
- Efficient workflow

## Best Practices

### Start Simple

- 3-5 main tags initially
- Add as needed
- Don't over-tag
- Quality over quantity

### Be Consistent

- Tag ideas immediately
- Use same tags for similar ideas
- Maintain color scheme
- Regular review

### Review Regularly

- Weekly tag audit
- Remove unused tags
- Update tag names
- Optimize organization
