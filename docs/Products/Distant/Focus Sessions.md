# Focus Sessions

Master the art of distraction-free focus.

## Starting a Session

### Setting Your Duration

**Circular Dial:**
- Touch and drag around the circle
- Precise control from 1-240 minutes
- Visual feedback as you adjust
- Release to set duration

**Quick Presets:**
- Tap preset buttons for instant selection
- Default: 15, 30, 60 minutes
- Customize in Settings → Customization
- Fastest way to start

**Duration Range:**
- Minimum: 1 minute
- Maximum: 240 minutes (4 hours)
- Or use Infinite Mode for unlimited time

### Choosing What to Focus On

1. Tap the focus type selector
2. Browse categories or search
3. Select your activity
4. Or create a custom focus type

### Optional: Block Apps

1. Tap menu → Block Apps
2. Select distracting apps
3. Apps will be blocked during session
4. Or use Device-Free Mode to block all apps

### Begin Focusing

1. Review your settings
2. Tap "Start Focus"
3. Apps are blocked immediately
4. Timer begins counting down

## During a Session

### Active Session Display

**Timer:**
- Shows remaining time
- Progress ring visualizes completion
- Can toggle seconds display in settings
- Clear, easy to read

**Progress Ring:**
- Fills as time progresses
- Color changes based on gradient settings
- Visual motivation
- Satisfying to watch complete

**Session Info:**
- Focus type displayed
- Duration shown
- Blocked apps indicator
- Current XP multipliers

### Session Controls

**Pause Button:**
- Temporarily stop the timer
- Paused time doesn't count
- Apps remain blocked
- Resume anytime

**Resume Button:**
- Continue from where you paused
- Timer picks up where it left off
- Seamless continuation

**Stop Button:**
- End session early
- Confirmation required
- Partial XP awarded
- Session saved to history

### Emergency Override

**When to Use:**
- True emergencies only
- Urgent phone call needed
- Critical message to send
- Unexpected situation

**How It Works:**
1. Tap "Emergency Override"
2. Hold for 10 seconds to confirm
3. Grants 5 minutes of app access
4. Apps automatically re-block after 5 minutes
5. Timer continues running

::: warning Use Sparingly
Emergency Override should be for genuine emergencies. Frequent use defeats the purpose of app blocking.
:::

## Overtime Mode

Continue focusing past your planned time.

### How Overtime Works

**When Timer Reaches Zero:**
1. Celebration animation plays
2. Timer switches to "Overtime"
3. Counts up from 00:00
4. Apps remain blocked
5. Continue focusing as long as you want

**Benefits:**
- Bonus XP for extra time
- Don't interrupt flow state
- Flexible completion
- Rewards dedication

**Completing Overtime:**
1. Tap "Complete Session" when ready
2. Review total time (planned + overtime)
3. Option to trim overtime
4. Earn XP for all time

### Managing Overtime

**In Session Feedback:**
- See planned vs actual time
- Trim overtime if desired
- Restore trimmed time
- Adjust for accurate tracking

**Why Trim:**
- Accidentally left timer running
- Took a break without pausing
- Want to track only active focus time
- Keep statistics accurate

## Pause & Resume

### When to Pause

**Good Reasons:**
- Bathroom break
- Get water/snack
- Answer urgent question
- Brief interruption

**Not Recommended:**
- Checking social media
- Long breaks
- Switching tasks
- Avoiding work

### How Pausing Works

**Pause:**
1. Tap "Pause Session"
2. Timer stops immediately
3. Apps remain blocked
4. Paused time doesn't count toward session

**Resume:**
1. Tap "Resume Session"
2. Timer continues from where it stopped
3. No time lost
4. Seamless continuation

**Paused State:**
- Timer shows "Paused"
- Duration preserved
- Apps still blocked
- Can resume anytime

## Stopping Early

### When to Stop

**Valid Reasons:**
- Completed your task early
- Unexpected urgent matter
- Not feeling well
- Need to switch activities

**How to Stop:**
1. Tap "Stop Session"
2. Confirm you want to stop
3. Session ends immediately
4. Partial XP awarded

**What Happens:**
- Apps are unblocked
- Session saved to history
- XP awarded for time completed
- Can rate and add notes

### Partial XP

**Calculation:**
- Based on time actually focused
- Bonuses still apply
- Proportional to completion
- Better than nothing!

**Example:**
- Planned: 60 minutes
- Completed: 30 minutes
- Earn XP for 30 minutes
- Plus any applicable bonuses

## Best Practices

### Before Starting

1. **Prepare workspace** - Clean, organized
2. **Have materials ready** - Everything you need
3. **Eliminate distractions** - Close unnecessary tabs
4. **Set clear goal** - Know what you'll work on
5. **Block apps** - Add your biggest distractions

### During Session

1. **Stay present** - Focus on the task
2. **Resist urges** - Don't reach for phone
3. **Use Pause wisely** - Only for necessary breaks
4. **Trust the process** - Let the timer guide you
5. **Embrace discomfort** - Focus gets easier with practice

### After Completing

1. **Rate honestly** - Helps you improve
2. **Add notes** - Capture insights
3. **Review progress** - Check analytics
4. **Celebrate** - Acknowledge your effort
5. **Plan next session** - Build momentum

## Troubleshooting

### Apps Not Blocking

**Solutions:**
1. Check Screen Time permission
2. Restart Distant
3. Restart device
4. Re-select apps to block
5. Ensure Screen Time is enabled in iOS Settings

### Session Not Restoring

**Solutions:**
1. Don't force-quit Distant during sessions
2. Keep app in background
3. Ensure sufficient storage
4. Update to latest version

### Timer Inaccurate

**Solutions:**
1. Don't change device time during session
2. Keep device charged
3. Avoid Low Power Mode during sessions
4. Report persistent issues to support
