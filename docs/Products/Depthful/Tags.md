# Tags & Organization

Organize your thoughts with powerful tagging and filtering.

## Default Tags

Depthful comes with 18 pre-configured tags, each with a unique color.

### Available Default Tags

**Reflection & Thinking:**
- **Thought** - General musings
- **Reflection** - Looking back
- **Daydream** - Imaginative thinking
- **Question** - Things you're wondering

**Goals & Growth:**
- **Goal** - Aspirations and targets
- **Milestone** - Achievements reached
- **Learning** - New knowledge
- **Advice** - Wisdom to remember

**Experiences:**
- **Experience** - Things that happened
- **Memory** - Moments to remember
- **Story** - Narratives and tales
- **Dream** - Nighttime dreams

**Emotions & Gratitude:**
- **Gratitude** - Things you're thankful for
- **Regret** - Things you wish were different
- **Inspiration** - What motivates you

**Ideas & Content:**
- **Idea** - Creative concepts
- **Quote** - Words worth saving
- **Journal** - Daily entries

### Tag Colors

Each default tag has a distinct color for easy visual identification:
- Quick recognition in lists
- Color-coded organization
- Visual categorization
- Consistent across app

## Custom Tags

Create unlimited custom tags for your specific needs.

### Creating Custom Tags

1. Tap **tags icon** in bottom bar
2. Tap **"Create Custom Tag"**
3. Enter tag name
4. Choose a color from the palette
5. Tap **"Save"**

### Editing Custom Tags

1. Go to tag management
2. Long press on custom tag
3. Select **"Edit"**
4. Change name or color
5. Save changes

### Deleting Custom Tags

1. Go to tag management
2. Long press on custom tag
3. Select **"Delete"**
4. Confirm deletion
5. Tag removed from all thoughts

::: warning Deleting Tags
Deleting a custom tag removes it from all thoughts that use it. This action cannot be undone.
:::

## Multi-Tag Support

Apply multiple tags to a single thought for flexible organization.

### Adding Multiple Tags

1. Open a thought
2. Tap tag icon or "Add Tags"
3. Select multiple tags
4. Tags appear on thought card
5. Use for cross-categorization

### Benefits of Multi-Tagging

**Better Organization:**
- Tag by type AND topic
- Example: "Idea" + "Work" + "Project Name"

**Flexible Filtering:**
- Find thoughts by any combination
- Example: All "Goals" that are also "Health"

**Rich Context:**
- Multiple perspectives on one thought
- Example: "Memory" + "Family" + "Gratitude"

### Multi-Tag Examples

**Work Idea:**
- Tags: Idea, Work, Project Alpha
- Find all project ideas easily

**Personal Goal:**
- Tags: Goal, Health, Fitness
- Track health-related goals

**Dream Journal:**
- Tags: Dream, Reflection, Question
- Analyze recurring dream themes

**Book Notes:**
- Tags: Learning, Quote, Inspiration
- Collect wisdom from reading

## Filtering & Finding

### Filter by Tags

**Single Tag Filter:**
1. Tap filter icon
2. Select one tag
3. See all thoughts with that tag
4. Clear to see all thoughts

**Multi-Tag Filter:**
1. Tap filter icon
2. Select multiple tags
3. See thoughts with ALL selected tags
4. Refine your search

### Recently Used Tags

- Recently used tags appear at top
- Quick access to frequent tags
- Speeds up tagging workflow
- Adapts to your habits

### Tag Search

1. In tag selection screen
2. Use search bar
3. Find tags by name
4. Faster than scrolling

## Tag Management

### Viewing All Tags

Access via tags icon in bottom bar:
- See all default tags
- See all custom tags
- View tag usage counts
- Manage custom tags

### Tag Usage Analytics

In Analytics view:
- See how often each tag is used
- Identify most-used tags
- Discover patterns
- Optimize tag system

### Tag Organization Tips

1. **Start with defaults** - Use built-in tags first
2. **Add custom gradually** - Create as needs arise
3. **Keep it simple** - Don't create too many tags
4. **Review regularly** - Delete unused custom tags
5. **Be consistent** - Use same tags for similar thoughts

## Best Practices

### Tagging Strategy

**Immediate Tagging:**
- Tag thoughts as you create them
- Easier than batch tagging later
- Builds good habits

**Consistent Usage:**
- Use same tags for similar content
- Develop personal conventions
- Makes filtering more effective

**Balanced Approach:**
- Not too many tags per thought (2-4 is good)
- Not too many custom tags (10-20 max)
- Quality over quantity

### Finding the Right Tags

**Ask Yourself:**
- What is this thought about?
- When might I want to find this?
- What category does it fit?
- What's the main theme?

**Common Combinations:**
- Type + Topic (Idea + Work)
- Type + Emotion (Memory + Gratitude)
- Topic + Project (Work + Client Name)
- Time + Type (2025 + Goal)

### Maintaining Your System

**Weekly Review:**
- Check tag usage in Analytics
- Identify underused tags
- Consider consolidating similar tags
- Delete unused custom tags

**Monthly Cleanup:**
- Review custom tag list
- Merge similar tags if needed
- Update tag colors if desired
- Refine your system

## Advanced Organization

### Tag Hierarchies

While Depthful doesn't have nested tags, you can create hierarchy with naming:
- "Work - Project A"
- "Work - Project B"
- "Personal - Health"
- "Personal - Finance"

### Tag Combinations

Create virtual categories with combinations:
- "Idea" + "Work" = Work ideas
- "Goal" + "Health" = Health goals
- "Memory" + "Family" = Family memories

### Seasonal Tags

Create custom tags for seasons or events:
- "Summer 2025"
- "Vacation"
- "Holiday Season"
- Review and archive after period ends
