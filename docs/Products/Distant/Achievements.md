## Viewing Achievements

### Achievement Screen

**Access:**
- Tap menu → Achievements
- See all achievements
- Locked and unlocked

**Display:**
- Unlocked achievements in color
- Locked achievements in gray
- Progress bars for multi-level achievements
- Unlock date shown

### Achievement Notifications

**When Unlocked:**
- In-app notification appears
- Celebration animation
- Achievement details shown
- Can be disabled in settings

**Notification Settings:**
- Settings → Customization → Notifications
- Toggle achievement notifications
- Choose notification sound

## Tips for Achievement Hunters

### Streak Achievements

**Maintaining Streaks:**
- Focus at least once per day
- Set daily reminders
- Even short sessions count
- Don't break the chain

**If You Miss a Day:**
- Start a new streak immediately
- Don't give up
- Learn from the miss
- Build back up

### Deep Focus Achievements

**Building Up:**
- Start with 30-minute sessions
- Gradually increase duration
- Work up to 60+ minutes
- Quality over quantity

**Staying Focused:**
- Block distracting apps
- Prepare workspace
- Have clear goals
- Take breaks between sessions

### Infinite Mode Achievements

**Getting Started:**
- Enable infinite mode
- Start with 30-minute goal
- See how long you can go
- Don't force it

**Long Sessions:**
- Well-rested and fed
- Clear schedule
- Engaging work
- Flow state activities

## Resetting Achievements

**Caution:**
- Settings → Clear Data → Clear All Achievements
- Permanently deletes achievement progress
- Cannot be undone
- Use only if starting fresh

**What Resets:**
- All achievement unlocks
- Achievement dates
- Progress toward achievements

**What Doesn't Reset:**
- Session history
- XP and levels
- Focus types
- Settings
