# Text & Watermarks

Add custom text overlays and watermarks to your mockups.

## Text Overlays

### Creating Text Layers

1. Click "Add Text Layer" button
2. Enter your text
3. Customize styling
4. Position using grid or manual coordinates
5. Adjust z-index for layering

### Font Selection

**20+ Google Fonts:**

**Sans-Serif (Modern, clean):**
- Inter, Lato, Source Sans Pro, Nunito
- Poppins, Raleway, Fira Sans, Ubuntu
- Work Sans, Quicksand, Barlow, DM Sans, Space Grotesk

**Serif (Traditional, elegant):**
- Merriweather, Lora, PT Serif, Crimson Text

**Monospace (Technical, code):**
- JetBrains Mono, Fira Code, Source Code Pro

### Text Styling

**Font Size:** 12-200px
- 12-24px: Captions
- 32-48px: Body text
- 64-96px: Headlines
- 128-200px: Large display

**Color:** Any color via picker

**Opacity:** 0-100%

**Formatting:**
- Bold, Italic, Underline
- Can combine multiple

**Alignment:**
- Left, Center, Right

### Advanced Features

**Text Shadow:**
- Enable for depth
- Shadow Color: Any color
- Shadow Blur: 0-20px
- Tips: 2-5px blur for subtle, 10-20px for glow

**Text Background:**
- Background box behind text
- Background Color: Any color
- Padding: 0-50px
- Use for readability over busy backgrounds

**Z-Index Control:**
- Bring to Front: Above images (z-index 10)
- Send to Back: Behind images (z-index 0)
- Use for watermarks or headlines

### Text Positioning

**9-Point Grid:**
- Top: Left, Center, Right
- Middle: Left, Center, Right
- Bottom: Left, Center, Right

**Manual Positioning:**
- X Position: 0-100%
- Y Position: 0-100%

### Managing Layers

- **Visibility:** Show/hide without deleting
- **Select:** Click to edit
- **Duplicate:** Copy with all settings
- **Delete:** Remove permanently
- **Reorder:** Use z-index controls

### Text Tips

**Headlines:** Large (80-120px), bold, centered

**Subheadings:** Medium (40-60px), regular weight

**Body Text:** Smaller (24-36px), left-aligned

**Watermarks:** Small (16-24px), low opacity (30-50%), behind images

## Watermarks

### Image Watermarks

**Uploading:**
1. Click "Upload Watermark"
2. Select logo/image (PNG recommended)
3. Adjust settings

**Controls:**
- Opacity: 0-100% (30-50% for subtle)
- Scale: 0.5-2.0x
- Position: 5 presets (corners + center)

### Text Watermarks

**Creating:**
1. Enter watermark text
2. Customize font, size, color
3. Adjust opacity and position

**Controls:**
- Text: Your watermark text
- Font: 20+ fonts
- Size: 12-72px
- Color: Any color
- Opacity: 0-100%
- Position: 5 presets

### Watermark Tips

- Subtle is better: 30-50% opacity
- Corner placement: Bottom-right is standard
- Don't overpower main content
- Use "Apply to All" for consistency

### Removing Watermarks

- Click "Remove Watermark"
- Upload new to replace
- Use "Clear All Watermarks" in Actions
