# Display Modes

Afternoon offers 6 different display modes to match your workflow and preferences.

## How to Switch Modes

Tap the **display mode selector** in the top bar to choose your view.

## 1. Combined View

**Default mode** - Shows habits and tasks together in chronological order by time.

**Features:**
- Items sorted by scheduled time
- Items without times appear at the bottom
- Mixed view of all your daily items
- Quick overview of your entire day

**Best for:**
- Seeing your entire day at a glance
- Understanding how habits and tasks fit together
- General daily use
- Beginners getting started

**Example:**
```
8:00 AM - Morning meditation (Habit)
9:00 AM - Team meeting (Task)
12:00 PM - Lunch walk (Habit)
2:00 PM - Finish report (Task)
```

## 2. Separated View

Displays habits and tasks in **separate sections**, each sorted by time.

**Features:**
- Clear distinction between habits and tasks
- Each section sorted chronologically
- Easier to focus on one type at a time
- Still see both on one screen

**Best for:**
- People who think of habits and tasks differently
- Reviewing one category at a time
- Maintaining separation between routines and to-dos

**Layout:**
```
HABITS
8:00 AM - Morning meditation
12:00 PM - Lunch walk

TASKS
9:00 AM - Team meeting
2:00 PM - Finish report
```

## 3. Habits Only

Shows **only your habits** for the selected date.

**Features:**
- Focused habit tracking view
- No task distractions
- See all habit details and progress
- Perfect for morning/evening routines

**Best for:**
- Dedicated habit tracking sessions
- Morning routine check-ins
- Evening habit reviews
- Building consistency without task noise

## 4. Tasks Only

Displays **only tasks** for the selected date.

**Features:**
- Pure task management view
- See carried-over tasks clearly
- Focus on getting things done
- No habit tracking distractions

**Best for:**
- Task-focused work sessions
- Planning your workday
- Reviewing what needs to be done
- GTD (Getting Things Done) workflow

## 5. Day Planner

**Timeline view** showing all items in a 24-hour schedule.

**Features:**
- 15-minute time increments
- Visual timeline from midnight to midnight
- Current time indicator (red line)
- Shows habits, tasks, time blocks, and calendar events
- Overlapping items shown side-by-side with horizontal scroll
- Tap checkbox to cycle completion states
- Double-tap to edit items
- Auto-scrolls to current time on today's view

**Best for:**
- Time-blocking and schedule visualization
- Seeing how your day flows
- Identifying gaps and overlaps
- Planning time-based activities
- Visual learners

**How it works:**
- Each hour is divided into 4 blocks (15 minutes each)
- Items appear at their scheduled times
- Duration shown visually for time blocks
- Current time marked with red line
- Scroll vertically through the day
- Scroll horizontally for overlapping items

::: tip Time Blocking
Day Planner is perfect for time-blocking your day. Create time blocks for focused work, breaks, and meetings to visualize your schedule.
:::

## 6. Calendar Only

Shows **only calendar events** from your Apple Calendar.

**Features:**
- Read-only view of calendar events
- See external commitments
- Event titles, times, and locations
- Tap to open in Calendar app

**Best for:**
- Viewing external commitments
- Checking meeting schedules
- Seeing what's on your calendar
- Planning around existing events

**Setup required:**
- Settings → Integrations → Calendar
- Grant calendar access

## Choosing the Right Mode

### By Time of Day

**Morning:**
- Habits Only - Focus on morning routine
- Combined View - See the day ahead

**During Work:**
- Tasks Only - Focus on getting things done
- Day Planner - Time-block your work

**Evening:**
- Habits Only - Complete evening routine
- Combined View - Review the day

### By Workflow

**Routine-Focused:**
- Habits Only
- Separated View

**Task-Focused:**
- Tasks Only
- Combined View

**Time-Blocking:**
- Day Planner
- Calendar Only (for reference)

**Balanced:**
- Combined View
- Separated View

### By Personality

**Visual Thinkers:**
- Day Planner
- Combined View

**List Lovers:**
- Separated View
- Tasks Only / Habits Only

**Minimalists:**
- Habits Only
- Tasks Only

**Planners:**
- Day Planner
- Calendar Only

## Switching Throughout the Day

You're not locked into one mode! Many users switch modes based on their current needs:

**Example Daily Flow:**
1. **Morning:** Habits Only - Complete morning routine
2. **Work Hours:** Day Planner - Time-block focused work
3. **Lunch:** Combined View - Quick check-in
4. **Afternoon:** Tasks Only - Power through to-dos
5. **Evening:** Habits Only - Evening routine

## Tips for Each Mode

### Combined View
- Set times for most items to maintain order
- Use descriptions to add context
- Expand items to see full details

### Separated View
- Complete habits first thing in the morning
- Review tasks after habits are done
- Use for weekly planning sessions

### Habits Only
- Perfect for building consistency
- Use during dedicated habit time
- Review streaks and progress

### Tasks Only
- Batch similar tasks together
- Set times to create structure
- Review carried-over tasks daily

### Day Planner
- Create time blocks for focused work
- Include breaks in your schedule
- Don't over-schedule - leave buffer time
- Use current time indicator to stay on track

### Calendar Only
- Check before planning your day
- Identify free time for tasks
- See external commitments at a glance
