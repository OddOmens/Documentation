# Getting Started

Create your first mockup in minutes.

## Creating Your First Mockup

### 1. Open FrameIt

Visit [frameit.social](https://frameit.social) in your web browser. No account or sign-up required.

### 2. Upload an Image

**Three ways to upload:**
- Click "Start Creating" button
- Drag and drop an image onto the canvas
- Click the upload icon in the gallery

**Supported formats:** PNG, JPEG, WebP

### 3. Customize

Use the control panels on the right to:
- Choose a layout
- Change background
- Add text overlays
- Apply effects
- Add watermarks

### 4. Export

Click the export button when satisfied with your design. Choose format and size, then download.

## Understanding the Interface

### Three Main Areas

**Left Sidebar (Gallery):**
- Manage multiple canvas designs
- Add new canvases
- Switch between designs
- Duplicate or delete canvases

**Center Canvas:**
- Real-time preview of your mockup
- See changes instantly
- Zoom in/out for detail work
- What you see is what you export

**Right Sidebar (Controls):**
- Layout selection
- Background customization
- Texture overlays
- Frame & shadow controls
- Text layers
- Watermarks
- Export options

## Image Requirements

### File Specifications

**Supported Formats:**
- PNG (recommended for transparency)
- JPEG
- WebP

**Size Limits:**
- Maximum file size: 10MB per image
- Minimum dimensions: 50×50 pixels
- Maximum dimensions: 8000×8000 pixels

**Tips:**
- Use high-resolution images for best quality
- PNG for graphics with transparency
- JPEG for photos
- Compress large files before uploading

## Quick Start Tips

### For Social Media Posts

1. Upload your screenshot or image
2. Choose "Single Image" layout
3. Select a gradient background
4. Add 50-75px padding
5. Export as 1080×1080 PNG

### For App Showcases

1. Upload multiple app screens
2. Choose "Three Columns" or "2×2 Grid"
3. Use solid color background
4. Add subtle shadows
5. Export as 1920×1080 PNG

### For Presentations

1. Upload your content images
2. Choose "Main + Sidebar" layout
3. Use professional gradient
4. Add text overlay for title
5. Export as 1920×1080 PNG

## Keyboard Shortcuts

**General:**
- `Ctrl/Cmd + Z` - Undo
- `Ctrl/Cmd + Shift + Z` - Redo
- `Ctrl/Cmd + S` - Save (auto-save enabled)

**Canvas:**
- `+` - Zoom in
- `-` - Zoom out
- `0` - Reset zoom to 100%

**Text Editing:**
- `Ctrl/Cmd + B` - Bold
- `Ctrl/Cmd + I` - Italic
- `Ctrl/Cmd + U` - Underline
