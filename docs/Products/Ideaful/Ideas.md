# Ideas & Tasks

Manage ideas and break them into actionable tasks.

## Ideas Management

### Creating Ideas

1. Tap **+** button
2. Enter details:
   - **Title** (required, 85 char max)
   - **Description** (optional, 85 char max)
   - **Status** (Unassigned default)
   - **Tags** (multiple allowed)
3. Auto-saves as you type

### Idea Components

**Title & Description:**
- Title: Main idea name
- Description: Short summary
- Character counters show remaining space
- Press Return to save and dismiss keyboard

**Status:**
- Current workflow stage
- Tap to change
- 7 default statuses available

**Tags:**
- Color-coded labels
- Multiple tags per idea
- Tap to add/remove
- Filter by tags

**Images:**
- Up to 10 per idea
- Inspiration and reference
- Tap to view full screen
- Long press to delete

**Notes:**
- Unlimited per idea
- Rich text support
- Auto-save
- Detailed documentation

**Tasks:**
- Action items
- Priorities and due dates
- Completion tracking
- Reminders

### Editing Ideas

- Tap any field to edit
- Changes save automatically
- Character counters update
- Press Return to dismiss keyboard

### Deleting Ideas

**Method 1:**
- Swipe left on idea
- Tap delete
- Confirm (optional)

**Method 2:**
- Open idea details
- Tap trash icon
- Confirm deletion

### Adding Images

1. Open idea details
2. Scroll to "Inspirational Images"
3. Tap **+** button
4. Select from photo library
5. Add up to 10 images

**Image Actions:**
- Tap to view full screen
- Long press to delete
- Context menu for options

## Task System

### Creating Tasks

**Quick Creation:**
1. Open idea
2. Tap list icon (top right)
3. Type in "New task..." field
4. Tap **+** to create

**Detailed Creation:**
1. Open idea
2. Tap list icon
3. Tap **+** button (empty field)
4. Fill in details:
   - Title
   - Description
   - Priority (None/Low/Medium/High)
   - Due Date
   - Reminder
5. Save

### Task Priorities

**High Priority (Red flag):**
- Urgent tasks
- Do first
- Red indicator

**Medium Priority (Orange flag):**
- Important tasks
- Do soon
- Orange indicator

**Low Priority (Yellow flag):**
- Nice-to-have
- Do when possible
- Yellow indicator

**No Priority (Gray):**
- Unassigned
- Default state

### Task Filtering

**Smart Filters:**
- **Today** - Due today (shows overdue count)
- **Tomorrow** - Due tomorrow (shows this week count)
- **High/Med/Low** - By priority level
- Tap to activate, tap again to clear

**Overview Cards:**
- **Task Overview** - Today and Tomorrow counts
- **Task Priorities** - High, Medium, Low counts
- Collapsible sections
- Quick reference

### Managing Tasks

**Toggle Completion:**
- Tap checkbox
- Marks complete/incomplete
- Updates counts

**Edit Task:**
- Tap task row
- Modify details
- Auto-saves

**Delete Task:**
- Swipe left
- Confirm deletion

**Clear Completed:**
- Tap trash icon in toolbar
- Removes all completed tasks
- Confirmation required

### All Tasks View

Access from Dashboard to see all uncompleted tasks:

**Features:**
- Tasks grouped by idea
- Same filtering options
- Tap idea name to jump
- Complete or edit directly

**Use Cases:**
- Daily task review
- Priority management
- Cross-idea task tracking
- Focus on what's due

### Task Notifications

**Setting Reminders:**
1. Create or edit task
2. Enable reminder
3. Set date and time
4. Save

**Notification Behavior:**
- Device notification at specified time
- Includes idea name and task title
- Requires notification permissions
- Prompted on first use

**Managing Notifications:**
- Settings → Features → Notifications
- Toggle device reminders
- Toggle due date notifications
- iOS Settings for permissions

## Notes

### Creating Notes

1. Open idea
2. Scroll to "Notes" section
3. Tap **+** button
4. Enter title and content
5. Tap **X** to save

### Note Features

- **Unlimited** notes per idea
- **Rich text** support
- **Auto-save** as you type
- **Timestamps** automatically recorded

### Editing Notes

- Tap note to open
- Edit title or content
- Changes save automatically
- Tap **X** to close

### Deleting Notes

- Swipe left on note
- Or tap trash icon
- Confirm deletion

### Note Organization

- List format display
- Title and preview (first 72 characters)
- Sorted by creation date
- Color-coded left border

## Best Practices

### Idea Capture

- Quick entry with just title
- Add details later
- Tag immediately
- Use descriptive titles

### Task Management

- Break down into small tasks
- Set realistic priorities
- Use due dates wisely
- Review daily

### Notes Usage

- Detailed documentation
- Research and links
- Multiple notes for aspects
- Regular updates

### Organization

- Consistent tagging
- Regular status updates
- Archive completed ideas
- Weekly reviews
