# Analytics & Insights

Track your writing patterns, streaks, and progress.

## Overview Statistics

Quick summary of your journaling journey.

### Total Thoughts
- Count of all thoughts created
- Includes favorites and regular thoughts
- Excludes deleted thoughts
- Lifetime total

### Words & Characters
- **Total words written** across all thoughts
- **Total characters** typed
- **Average word count** per thought
- Measures your writing volume

### Most Active Day
- Day with most thoughts created
- Shows entry count for that day
- Identifies peak productivity
- Motivational milestone

## Activity Tracking

Visualize your writing patterns over time.

### Time Ranges

**7 Days:**
- Last week's activity
- Daily thought count
- Recent patterns
- Short-term trends

**30 Days:**
- Last month's activity
- Weekly patterns
- Monthly consistency
- Medium-term view

**Previous Year:**
- Full year of data
- Seasonal patterns
- Long-term trends
- Year-over-year comparison

**All Time:**
- Complete history
- Lifetime patterns
- Overall consistency
- Full journey view

### Activity Charts

**Visual Display:**
- Bar charts showing thought count
- Daily, weekly, or monthly views
- Color-coded activity levels
- Easy pattern recognition

**Insights:**
- Identify most productive days
- See writing streaks
- Notice gaps
- Track consistency

## Writing Streaks

Build momentum with consecutive writing days.

### Current Streak

**What It Is:**
- Consecutive days with at least one thought
- Resets if you miss a day
- Motivates daily writing
- Builds habit

**How It Works:**
- Write at least one thought per day
- Streak increases by 1
- Miss a day, streak resets to 0
- Start building again

### Longest Streak

**Personal Record:**
- Your best streak ever
- Goal to beat
- Achievement milestone
- Proof of consistency

**Tracking:**
- Automatically recorded
- Never resets
- Shown in analytics
- Celebrate when you beat it

### Streak Tips

::: tip Building Streaks
- Write something every day, even if short
- Set a daily reminder
- Write at the same time
- Don't break the chain!
:::

**Maintaining Streaks:**
- Keep it simple on busy days
- One sentence counts
- Use voice recordings for speed
- Make it a habit

## Using Analytics Effectively

### Daily Check-In

**Morning:**
- Check current streak
- Motivate daily writing
- Set intention

**Evening:**
- Review today's thoughts
- Maintain streak
- Reflect on day

### Weekly Review

**Every Week:**
1. Check 7-day activity
2. Review tag usage
3. Note patterns
4. Adjust habits

**Questions to Ask:**
- Am I writing consistently?
- Which tags am I using most?
- What themes emerge?
- How can I improve?

### Monthly Analysis

**Every Month:**
1. Review 30-day activity
2. Check longest streak
3. Analyze tag patterns
4. Set goals for next month

**Insights to Gather:**
- Writing frequency trends
- Topic patterns
- Consistency improvements
- Areas to explore

### Yearly Reflection

**End of Year:**
1. Review all-time stats
2. Check previous year data
3. Celebrate milestones
4. Plan for next year

**Celebrate:**
- Total thoughts written
- Longest streak achieved
- Words written
- Growth and consistency

## Setting Goals

### Streak Goals

**Start Small:**
- 7-day streak
- 14-day streak
- 30-day streak
- 100-day streak

**Build Up:**
- Increase gradually
- Celebrate milestones
- Don't give up after breaks
- Focus on consistency

### Writing Volume Goals

**Daily:**
- Write at least one thought
- Minimum word count
- Include specific tags
- Add media

**Weekly:**
- 7 thoughts minimum
- Explore different tags
- Try voice recordings
- Add photos

**Monthly:**
- 30 thoughts (one per day)
- Use all default tags once
- Create custom tags
- Export for backup

### Content Goals

**Diversity:**
- Use variety of tags
- Mix text, voice, photos
- Different thought types
- Explore new topics

**Depth:**
- Increase average word count
- More detailed reflections
- Longer thoughts
- Richer content
