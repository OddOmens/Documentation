# Getting Started

Start your journey to better focus with Distant.

## Installation & Setup

1. **Download Distant** from the App Store
2. **Open the app** for the first time
3. **Grant Screen Time permission** when prompted (required for app blocking)
4. **Grant notification permission** (optional but recommended)

### Screen Time Permission

**Why it's needed:**
- Enables app blocking during focus sessions
- Uses Apple's official Screen Time API
- Secure and privacy-focused

**How to grant:**
1. iOS will prompt on first use
2. Or go to Settings → Screen Time → Content & Privacy Restrictions
3. Enable Screen Time if disabled
4. Grant Distant permission

## Your First Focus Session

### Quick Start

1. **Set duration** - Drag the circular dial or tap a preset (15, 30, 60 min)
2. **Choose focus type** - Tap to select (Reading, Working, etc.)
3. **Tap "Start Focus"** - Begin your session
4. **Stay focused** - Timer counts down
5. **Complete** - Celebrate your achievement!

::: tip Start Small
Begin with 15-30 minute sessions. Build up to longer durations as your focus muscle strengthens.
:::

## Understanding the Interface

### Main Screen

**Circular Dial:**
- Drag around the circle to set duration
- Shows selected time in center
- Visual progress ring during sessions
- Smooth, intuitive control

**Preset Buttons:**
- Quick duration selection
- Default: 15, 30, 60 minutes
- Customizable in settings
- Tap for instant selection

**Focus Type Selector:**
- Shows current focus type
- Tap to change
- 30+ built-in types
- Create custom types

**Infinite Mode Toggle:**
- Below focus type selector
- Enable for unlimited focus time
- Timer counts up instead of down
- Perfect for flow state work

**Start Button:**
- Large, prominent button
- Tap to begin session
- Changes to Pause/Resume during session
- Stop button appears when active

### During a Session

**Timer Display:**
- Shows remaining time
- Progress ring visualizes completion
- Can show/hide seconds in settings
- Overtime mode when time expires

**Control Buttons:**
- **Pause** - Temporarily pause session
- **Resume** - Continue from where you paused
- **Stop** - End session early
- **Emergency Override** - Temporary app access (hold 10 seconds)

## Basic Operations

### Setting Duration

**Using the Dial:**
1. Touch and drag around the circle
2. Release when desired time is shown
3. Precision control for any duration
4. 1-240 minutes available

**Using Presets:**
1. Tap preset button (15, 30, or 60)
2. Instant duration selection
3. Customize presets in settings
4. Fastest way to start

### Choosing Focus Type

**Built-in Types:**
- Personal Development: Reading, Learning, Studying
- Work: Working, Writing, Coding, Planning
- Creative: Creating, Designing, Music, Drawing
- Wellness: Meditation, Yoga, Exercise, Breathing
- And many more...

**Selecting:**
1. Tap focus type selector
2. Browse categories
3. Tap to select
4. Returns to main screen

### Blocking Apps

**First Time Setup:**
1. Tap menu → Block Apps
2. Browse installed apps
3. Tap to select apps to block
4. Selected apps show checkmark
5. Tap Done

**What Gets Blocked:**
- Social media apps
- Messaging apps
- Games
- Any app you select
- Blocks persist even if Distant is closed

**Device-Free Mode:**
- Special focus type
- Blocks ALL apps except essentials
- Only Phone, Messages, Settings remain
- Ultimate distraction elimination

## Completing Your First Session

### Normal Completion

1. Timer reaches zero
2. Celebration animation plays
3. Session feedback screen appears
4. Rate your session (1-5 stars)
5. Add notes (optional)
6. Earn XP and check achievements

### Overtime Mode

**What is it:**
- Continue focusing past planned time
- Timer switches to counting up
- Earn bonus XP
- Complete when ready

**How to use:**
1. Timer reaches zero
2. Keep focusing
3. Tap "Complete Session" when done
4. Review overtime in feedback
5. Trim if desired

### Session Feedback

**Rate Your Session:**
- 1-5 stars
- Track what works
- Review later in history

**Add Notes:**
- What you accomplished
- How you felt
- What helped/hindered
- Future improvements

**Overtime Options:**
- Keep all overtime
- Trim to planned duration
- Restore if you change mind

## Tips for Beginners

### Building the Habit

1. **Start with one session daily**
2. **Same time each day** - Morning or evening
3. **Same focus type** - Build consistency
4. **Track your streak** - Don't break the chain

### Maximizing Success

**Before Starting:**
- Prepare your workspace
- Have materials ready
- Use the bathroom
- Get water/coffee
- Silence other devices

**During Session:**
- Keep Distant running
- Don't force-quit the app
- Use Pause for breaks
- Emergency Override only for emergencies

**After Completing:**
- Rate honestly
- Add notes
- Check progress
- Celebrate!

### Common Mistakes to Avoid

1. **Starting too long** - Begin with 15-30 minutes
2. **Not blocking apps** - Block your biggest distractions
3. **Force-quitting app** - Keep it running in background
4. **Skipping feedback** - Ratings help you improve
5. **Giving up after one miss** - Consistency over perfection
