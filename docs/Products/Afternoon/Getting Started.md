# Getting Started

Get up and running with Afternoon in minutes.

## Installation

1. **Download Afternoon** from the App Store
2. **Open the app** and begin the onboarding process
3. **Grant permissions** for the best experience (optional)

## First Launch Onboarding

When you first open Afternoon, you'll go through an 8-step setup:

### 1. Welcome Screen
Introduction to the app and its capabilities.

### 2. Name Input
Enter your name (or continue as "Habitor"). This appears in your daily greeting.

### 3. Color Selection
- Choose your accent color
- Select light or dark mode
- These can be changed anytime in settings

### 4. Interaction Guide
Learn the tap gestures:
- **Single tap:** Expand item details
- **Double tap:** Open item for editing

### 5. Notifications
Grant permission for reminders (recommended for habit tracking).

### 6. Wake Time
Set your morning start time - this appears at the top of your daily view.

### 7. Sleep Time
Set your evening end time - this appears at the bottom of your daily view.

### 8. Habit Selection
Choose from pre-made habit templates to get started quickly, or skip to create your own.

## Quick Add Bar

The fastest way to create items in Afternoon:

**Location:** Bottom of the main screen

**How to use:**
1. Tap the text field to start typing
2. Select the item type:
   - Task
   - Habit
   - Reminder
   - Time Block
   - Calendar Event
3. Type the name and press return
4. The item is created instantly

::: tip Quick Creation
Use Quick Add for rapid capture. You can always double-tap the item later to edit details like time, description, and reminders.
:::

## Main Screen Layout

### Top Bar
- **Profile picture** - Tap to open your profile
- **Greeting** - Time-based greeting ("Good Morning, [Name]")
- **Display mode selector** - Switch between view modes
- **Menu** - Three dots for settings and features

### Calendar Strip
- 7-day week view with progress circles
- Swipe or tap to change dates
- Current day highlighted in your accent color
- Progress circles show daily completion percentage

### Content Area
- Displays habits, tasks, and other items
- Swipe left/right to navigate between days
- Pull down to refresh

### Date Selector
- Shows current or selected date
- "Today" button appears when viewing other dates
- Tap to open date picker for quick navigation

## Creating Your First Habit

**Method 1: Quick Add**
1. Tap the Quick Add bar at the bottom
2. Type your habit name (e.g., "Morning meditation")
3. Select "Habit" from the type selector
4. Press return

**Method 2: Menu**
1. Tap menu (three dots) → Manage Habits
2. Tap the + button
3. Fill in details:
   - Title (required)
   - Description (optional)
   - Start date
   - Time (optional but recommended)
   - Goal tracking (optional)
   - Reminders (optional)
4. Tap Save

## Creating Your First Task

**Method 1: Quick Add**
1. Tap the Quick Add bar
2. Type your task name
3. Select "Task"
4. Press return

**Method 2: Menu**
1. Tap menu → Manage Tasks
2. Tap the + button
3. Fill in details:
   - Title (required)
   - Description (optional)
   - Date (required)
   - Time (optional)
   - Notification (optional)
4. Tap Save

## Completing Items

### Habits
Tap the checkbox to cycle through states:
- Empty (incomplete)
- Checked (completed)
- Ring (skipped)

### Tasks
Tap the dashed checkbox to toggle:
- Empty (incomplete)
- Checked (completed)

## Recommended Settings

For the best experience:

1. **Enable iCloud Sync**
   - iOS Settings → [Your Name] → iCloud
   - Keeps your data backed up and synced

2. **Allow Notifications**
   - Set reminders for new habits
   - Get notified about achievements

3. **Set Wake/Sleep Times**
   - Helps organize your day
   - Provides structure to your schedule

4. **Choose Your Display Mode**
   - Try different modes to find what works for you
   - Combined View is great for beginners

## Tips for Beginners

::: tip Start Small
Begin with 3-5 habits you want to build. Don't try to change everything at once.
:::

- **Set specific times** for habits to increase completion rates
- **Enable reminders** for new habits until they become automatic
- **Use the skip feature** for rest days instead of breaking streaks
- **Check analytics weekly** to see your progress
- **Explore display modes** to find your preferred view
