# Getting Started

Start capturing and organizing your ideas.

## First Launch

When you first open Ideaful:
- Dashboard view shows ideas grouped by status
- iCloud sync automatically enabled
- Ready to create your first idea

## Creating Your First Idea

1. Tap **+** button (top right)
2. Enter **Title** (required, max 85 characters)
3. Add **Description** (optional, max 85 characters)
4. Select **Status** (defaults to "Unassigned")
5. Add **Tags** for organization
6. Saves automatically as you type

## Understanding the Interface

### Dashboard

**Main View:**
- Ideas grouped by status
- Expandable status sections
- Count badges show ideas per status
- Empty statuses hidden automatically

**Toolbar:**
- **+** - Create new idea
- **â˜°** - Menu (bulk operations, settings)
- **Tag icon** - Filter by tags
- **List icon** - View all tasks

### Idea Details

**Sections:**
- Title and description
- Status selector
- Tags
- Inspirational images (up to 10)
- Notes (unlimited)
- Tasks (with priorities and due dates)

### Navigation

- **Tap** idea to view details
- **Swipe left** to delete
- **Swipe right** for quick actions (Complete, Start, Hold)
- **Long press** for context menu

## Quick Actions

### Swipe Gestures

**Swipe Left:**
- Delete idea
- Confirmation dialog (optional)

**Swipe Right:**
- Complete
- Start
- Hold
- Quick status changes

### Keyboard Shortcuts

- **Return/Enter** - Save and dismiss keyboard
- **Swipe down** - Dismiss keyboard
- **Tab** - Move between fields (external keyboard)

## Basic Workflows

### Capture an Idea

1. Tap **+**
2. Enter title
3. Add tags
4. Save (automatic)
5. Add details later

### Develop an Idea

1. Open idea
2. Add description
3. Upload inspiration images
4. Create notes
5. Break into tasks
6. Set status to "Planning"

### Execute an Idea

1. Set status to "Started"
2. Complete tasks one by one
3. Update notes with progress
4. Mark tasks complete
5. Set status to "Completed"

## Tips for Beginners

### Start Simple

- Create a few ideas first
- Add details gradually
- Explore features as needed
- Don't overthink organization

### Use Tags Early

- Create 3-5 main tags
- Tag ideas immediately
- Use consistent colors
- Filter to focus

### Break Down Ideas

- Create tasks for action items
- Set priorities
- Add due dates
- Track progress

### Regular Review

- Weekly: Review all ideas
- Update statuses
- Complete tasks
- Archive finished ideas
