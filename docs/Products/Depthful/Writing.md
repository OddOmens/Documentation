# Writing & Formatting

Master rich text formatting and markdown in Depthful.

## Markdown Support

Depthful supports markdown formatting for rich, expressive writing.

### Basic Formatting

**Bold Text:**
- Markdown: `**text**`
- Toolbar: Tap **B** button
- Result: **bold text**

**Italic Text:**
- Markdown: `*text*`
- Toolbar: Tap *I* button
- Result: *italic text*

**Underline Text:**
- Markdown: `_text_`
- Toolbar: Tap U button
- Result: <u>underlined text</u>

**Code/Monospace:**
- Markdown: `` `text` ``
- Toolbar: Tap `<>` button
- Result: `monospace text`

**Strikethrough:**
- Markdown: `~text~`
- Toolbar: Tap ~S~ button
- Result: ~~strikethrough text~~

### Links

**Creating Links:**
- Markdown: `[Link Text](https://example.com)`
- Toolbar: Tap link button, enter URL and text
- Result: Clickable link

**Example:**
```markdown
Check out [Odd Omens](https://oddomens.com) for more apps!
```

## Writing Tips

### Structuring Your Thoughts

**Use Headers (in markdown):**
```markdown
# Main Topic
## Subtopic
### Detail
```

**Use Lists:**
```markdown
- Point one
- Point two
- Point three

1. First item
2. Second item
3. Third item
```

**Use Emphasis:**
- **Bold** for important points
- *Italic* for emphasis or thoughts
- `Code` for technical terms or quotes

### Writing Styles

**Daily Journal:**
```markdown
**Morning Reflection**
Woke up feeling energized today. 
Goals for the day:
- Finish project proposal
- Exercise for 30 minutes
- Call mom

**Evening Thoughts**
Accomplished most goals. Feeling proud!
```

**Idea Capture:**
```markdown
**App Idea: Meal Planner**
- Weekly meal planning
- Shopping list generation
- Recipe database
- Nutrition tracking

*Next steps:* Research competitors
```

**Dream Journal:**
```markdown
**Dream - November 12, 2025**
I was flying over a city made of glass...
~Felt like freedom~
*Possible meaning:* Desire for escape?
```

**Gratitude Practice:**
```markdown
**3 Things I'm Grateful For:**
1. Morning coffee with a friend
2. Sunny weather
3. Finished a challenging task
```

## Auto-Save

### How It Works

- Thoughts save automatically as you type
- No manual save button needed
- Changes persist immediately
- Safe to close app anytime

### Empty Thoughts

- Thoughts with no content are automatically discarded
- Prevents clutter in your journal
- Add at least one character to save

## Timestamps

### Automatic Timestamps

**Created:**
- Recorded when thought is first saved
- Never changes
- Shown in thought editor

**Last Updated:**
- Updates every time you edit
- Tracks most recent change
- Shown in thought editor

### Using Timestamps

- Track when ideas occurred
- See how thoughts evolve
- Identify active writing periods
- Review in Analytics

## Preview Mode

### Viewing Formatted Text

While editing:
1. Markdown renders in real-time
2. See formatting as you type
3. Links become clickable
4. Clean, readable display

### Edit vs Preview

- **Edit mode:** See markdown syntax
- **Preview mode:** See rendered result
- Toggle between modes as needed
- Both show formatting

## Best Practices

### For Clarity

1. **Use formatting sparingly** - Don't overdo it
2. **Bold key points** - Make important info stand out
3. **Use lists** - Organize related items
4. **Add links** - Reference sources or related content

### For Organization

1. **Start with a header** - Title your thought
2. **Use sections** - Break long thoughts into parts
3. **Add context** - Include dates, locations, people
4. **Tag appropriately** - Help future you find it

### For Consistency

1. **Develop a style** - Use formatting consistently
2. **Create templates** - For recurring thought types
3. **Use custom tags** - For your specific needs
4. **Review regularly** - See what works

## Common Formatting Patterns

### Meeting Notes

```markdown
**Meeting: Project Kickoff**
*Date:* November 12, 2025
*Attendees:* Alice, Bob, Carol

**Key Points:**
- Project timeline: 3 months
- Budget approved
- Next meeting: November 19

**Action Items:**
- [ ] Draft proposal (me)
- [ ] Review budget (Alice)
```

### Book Notes

```markdown
**Book:** *Atomic Habits*
**Author:** James Clear

**Key Takeaways:**
1. Small changes compound over time
2. Focus on systems, not goals
3. Make good habits obvious and easy

**Favorite Quote:**
> "You do not rise to the level of your goals. 
> You fall to the level of your systems."
```

### Personal Reflection

```markdown
**Reflection on Today**

**What went well:**
- Productive morning work session
- Good conversation with friend

**What could improve:**
- Spent too much time on social media
- Skipped evening workout

**Tomorrow's focus:**
Be more intentional with time
```

- **Italic:** Cmd + I
- **Undo:** Cmd + Z
- **Redo:** Cmd + Shift + Z

### Quick Actions

- **New line:** Return key
- **Indent:** Tab (in lists)
- **Autocomplete:** Tap suggestions
