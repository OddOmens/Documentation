# Time Blocking & Day Planner

Plan your day visually with time blocks and timeline view.

## Day Planner View

A visual timeline showing your entire day from midnight to midnight.

### How to Access

Tap the **display mode selector** → Choose **Day Planner**

### Features

- **24-hour timeline** with 15-minute increments
- **Current time indicator** (red line)
- **All item types** - habits, tasks, time blocks, calendar events
- **Overlapping items** shown side-by-side
- **Horizontal scroll** for multiple items at same time
- **Tap checkbox** to cycle completion states
- **Double-tap** to edit items
- **Auto-scroll** to current time on today's view

### Timeline Layout

**Time Increments:**
- Each hour divided into 4 blocks (15 minutes each)
- Vertical scroll through the day
- Horizontal scroll for overlapping items

**Visual Elements:**
- **Red line** - Current time
- **Colored blocks** - Time blocks with type colors
- **Checkboxes** - Habits and tasks
- **Calendar icon** - Calendar events
- **Duration bars** - Show length of time blocks

## Time Blocks

Allocate specific time periods for focused work, breaks, and activities.

### Creating Time Blocks

**Via Quick Add:**
1. Tap Quick Add bar
2. Type block title
3. Select "Time Block"
4. Press return
5. Double-tap to set times and type

**Via Menu:**
1. Menu → Create Time Block
2. Fill in details:
   - **Title** - What you'll do
   - **Start time** - When it begins
   - **End time** - When it ends
   - **Block type** - Category
   - **Color** - Visual identifier
   - **Notes** - Additional context

### Time Block Types

Each type has a unique icon and suggested color:

- **Focus** - Deep work, concentration
- **Break** - Rest, recharge
- **Meeting** - Calls, discussions
- **Exercise** - Physical activity
- **Learning** - Study, courses
- **Creative** - Writing, design, art
- **Admin** - Email, paperwork
- **Social** - Networking, socializing
- **Personal** - Self-care, hobbies

### Time Block Features

**Duration Display:**
- Visual bar shows block length
- Start and end times labeled
- Easy to see time allocation

**Completion Tracking:**
- Checkbox to mark block as used
- Track adherence to schedule
- See which blocks you complete

**Flexible Editing:**
- Adjust times as needed
- Change block type
- Update notes
- Delete if plans change

## Time Blocking Strategies

### Deep Work Blocks

**Setup:**
- 90-120 minute Focus blocks
- No interruptions
- Single task or project
- Breaks between blocks

**Example:**
```
9:00-10:30 AM - Focus: Write report
10:30-10:45 AM - Break
10:45-12:15 PM - Focus: Code feature
```

### Energy-Based Scheduling

**Match tasks to energy levels:**
- **Morning** - Creative, complex work
- **Midday** - Meetings, collaboration
- **Afternoon** - Admin, routine tasks
- **Evening** - Planning, light work

### Theme Days

**Dedicate days to specific work:**
- Monday - Planning & Admin
- Tuesday - Deep Work
- Wednesday - Meetings
- Thursday - Creative Projects
- Friday - Review & Cleanup

### Time Batching

**Group similar activities:**
- Email block (30 min, 2x daily)
- Meeting block (afternoons)
- Admin block (Friday mornings)
- Learning block (evenings)

## Tips for Successful Time Blocking

### Getting Started

1. **Start simple** - Block 2-3 focus periods
2. **Include breaks** - Don't schedule every minute
3. **Be flexible** - Adjust as needed
4. **Review daily** - Learn and improve

### Best Practices

1. **Block similar tasks** - Batch related work
2. **Protect focus time** - Guard deep work blocks
3. **Schedule breaks** - Rest is productive
4. **Leave buffer time** - For unexpected items
5. **Review and adjust** - Improve your system

### Common Mistakes

**Over-scheduling:**
- Don't fill every minute
- Leave 20-30% unscheduled
- Allow for flexibility

**Ignoring energy:**
- Match tasks to energy levels
- Don't schedule hard work when tired
- Use natural rhythms

**No breaks:**
- Include rest periods
- 5-15 minutes between blocks
- Longer breaks for meals

**Too rigid:**
- Allow adjustments
- Don't stress about perfection
- Adapt to reality

### Advanced Techniques

**Pomodoro Integration:**
- 25-minute focus blocks
- 5-minute breaks
- 4 pomodoros, then long break

**Time Boxing:**
- Set strict time limits
- Move on when time expires
- Prevents perfectionism

**Themed Blocks:**
- Color-code by project
- Visual project time allocation
- Easy to see balance

**Weekly Planning:**
- Plan blocks for entire week
- Recurring blocks for routines
- Adjust daily as needed

## Troubleshooting

### Overlapping Items

**Problem:** Multiple items at same time

**Solution:**
- Horizontal scroll to see all
- Adjust times to prevent overlap
- Use different item types strategically

### Can't See Current Time

**Problem:** Red line not visible

**Solution:**
- Scroll to current time
- Only shows on today's view
- Check you're viewing today

### Blocks Not Showing

**Problem:** Time blocks not appearing

**Solution:**
- Ensure you're in Day Planner view
- Check block has start/end times
- Verify correct date selected
