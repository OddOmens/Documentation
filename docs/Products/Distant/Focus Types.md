## Custom Focus Types

Create unlimited custom focus types tailored to your needs.

### Creating Custom Types

1. Tap menu → Focus Types
2. Tap "+" to create new
3. Enter name
4. Choose icon (93 available)
5. Select color (13 options)
6. Tap "Save"

### Icon Categories

**93 icons across 10 categories:**
- Work & Productivity
- Creative & Arts
- Health & Wellness
- Learning & Education
- Entertainment
- Social & Communication
- Home & Personal
- Sports & Fitness
- Nature & Outdoors
- Miscellaneous

### Color Options

**13 colors available:**
- Blue, Green, Purple, Red
- Orange, Yellow, Pink, Teal
- Indigo, Mint, Brown, Gray, Black

### Editing Custom Types

1. Go to Focus Types
2. Tap custom type
3. Edit name, icon, or color
4. Save changes
5. Updates apply to future sessions

### Deleting Custom Types

1. Go to Focus Types
2. Swipe left on custom type
3. Tap "Delete"
4. Confirm deletion
5. Past sessions preserve the type name

## Using Focus Types

### Selecting a Type

**Before Session:**
1. Tap focus type selector on main screen
2. Browse or search
3. Tap to select
4. Returns to main screen

**During Session:**
- Cannot change focus type
- Locked for session duration
- Ensures accurate tracking

### Why Use Focus Types

**Benefits:**
- Track time per activity
- See which activities you focus on most
- Level up in specific areas
- Earn category-specific achievements
- Analyze productivity patterns

### Focus Type Levels

**Individual Progression:**
- Each focus type has its own level
- Earn XP for that type when you use it
- 6 levels: Beginner → Master
- Track mastery in each area

**Level Names (Customizable):**
- Traditional: Beginner, Novice, Intermediate, Advanced, Expert, Master
- Competitive: Bronze, Silver, Gold, Platinum, Diamond, Obsidian
- Royalty: Peasant, Commoner, Noble, Lord, King, Monarch
- Nature: Seed, Sprout, Sapling, Tree, Redwood, Ancient Oak
- Zen: Seeker, Student, Practitioner, Teacher, Master, Enlightened
- Gaming: Noob, Casual, Regular, Veteran, Pro, Legend

## Device-Free Mode

The ultimate distraction elimination.

### What It Does

**Blocks Everything:**
- All apps except essentials
- Only Phone, Messages, Settings remain
- Complete digital detox
- Maximum focus

### When to Use

**Perfect For:**
- Deep work sessions
- Important studying
- Creative flow state
- Digital detox
- Breaking phone addiction

### How to Use

1. Select "Device-Free" as focus type
2. Set duration
3. Start session
4. Only essential apps accessible

## Best Practices

### Choosing the Right Type

**Be Specific:**
- "Coding" instead of "Working"
- "Reading" instead of "Learning"
- Helps with analytics

**Consistent Usage:**
- Use same type for same activities
- Builds accurate statistics
- Better progress tracking

### Custom Type Ideas

**Work Projects:**
- Client names
- Project names
- Specific tasks

**Learning:**
- Course names
- Subject areas
- Skill development

**Creative:**
- Specific mediums
- Project types
- Creative practices

**Personal:**
- Hobbies
- Side projects
- Personal development

### Organization Tips

1. **Don't create too many** - Keep it manageable
2. **Use built-ins first** - Cover most needs
3. **Create as needed** - Add when you have a specific use case
4. **Review regularly** - Delete unused custom types
5. **Be consistent** - Use same types for same activities
