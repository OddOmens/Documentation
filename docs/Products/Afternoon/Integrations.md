# Integrations

Connect Afternoon with Apple Health and Calendar for enhanced functionality.

## How to Access

Menu → Settings → Integrations

## Apple Health Integration

Sync your habit data with Apple Health automatically.

### Supported Categories

**Water Intake:**
- Track water consumption
- Syncs to Health as milliliters
- Habit: "Drink water" with Glasses unit

**Mindfulness:**
- Track meditation and breathing
- Syncs as Mindful Minutes
- Habit: "Meditate" with Minutes unit

**Workouts:**
- Track exercise sessions
- Syncs with activity type
- Habit: "Exercise" with Minutes/Sessions

**Sleep:**
- Track sleep duration
- Syncs as Sleep Analysis
- Habit: "Sleep" with Hours/Slept unit

**Steps:**
- Track walking/running
- Syncs step count
- Habit: "Walk" with Steps unit

**Heart Rate:**
- Track heart rate data
- Syncs BPM readings
- Habit: Heart rate tracking

### Smart Matching

Afternoon automatically suggests Health categories based on habit names:

**Examples:**
- "Drink water" → Water Intake
- "Meditate" → Mindfulness
- "Run" → Workouts (Running)
- "Yoga" → Workouts (Yoga)
- "Sleep" → Sleep Analysis
- "Walk" → Steps

### Setup

1. **Grant Permissions:**
   - Settings → Integrations → Apple Health
   - Tap "Connect to Health"
   - Select categories to sync
   - Grant read/write permissions

2. **Configure Habits:**
   - Edit habit
   - Tap "Health Category"
   - Select appropriate category
   - Choose activity type (for workouts)

3. **Start Tracking:**
   - Complete habit as normal
   - Data syncs automatically to Health
   - View in Health app

### How It Works

**For Simple Habits:**
- Completion syncs as 1 session/occurrence
- Timestamp recorded
- Appears in Health app

**For Goal-Based Habits:**
- Tracked amount syncs to Health
- Example: 30 minutes meditation → 30 Mindful Minutes
- Units converted automatically

**Sync Timing:**
- Syncs when habit is completed
- Immediate sync to Health
- No manual sync required

### Supported Units

**Water Intake:**
- Glasses → Milliliters
- Configurable glass size

**Mindfulness:**
- Minutes → Seconds (converted automatically)
- Duration tracked

**Workouts:**
- Minutes → Duration
- Activity type included
- Calories estimated

**Sleep:**
- Hours → Sleep duration
- Slept unit → Sleep Analysis

### Workout Types

When syncing exercise habits, choose from:
- Running
- Walking
- Cycling
- Swimming
- Yoga
- Strength Training
- HIIT
- Dance
- And more...

### Privacy & Data

**Your Control:**
- You choose what to sync
- Can revoke permissions anytime
- Data stored in your Health app
- Afternoon doesn't access other Health data

**Data Flow:**
- One-way sync: Afternoon → Health
- Afternoon doesn't read Health data
- Your habit data only
- Secure and private

### Troubleshooting

**Data Not Syncing:**
1. Check Health permissions
2. Verify habit has Health category selected
3. Ensure habit is completed (not skipped)
4. Check Health app for data

**Wrong Category:**
1. Edit habit
2. Change Health category
3. Future completions use new category
4. Past data remains unchanged

**Duplicate Data:**
- Avoid creating same habit twice
- Check Health app for duplicates
- Revoke and re-grant permissions if needed

## Apple Calendar Integration

View your calendar events alongside habits and tasks.

### Features

**Read-Only Access:**
- View calendar events
- No modifications to calendar
- Safe integration

**Event Display:**
- Event title
- Start and end times
- Location (if available)
- Calendar icon for identification

**Where Events Appear:**
- Day Planner view
- Calendar Only view
- Integrated with your schedule

### Setup

1. **Grant Permission:**
   - Settings → Integrations → Calendar
   - Tap "Connect to Calendar"
   - Grant calendar access when prompted

2. **View Events:**
   - Switch to Day Planner view
   - Or select Calendar Only display mode
   - Events load automatically

3. **Interact with Events:**
   - Tap event to open in Calendar app
   - View details in Afternoon
   - Plan around commitments

### How It Works

**Event Loading:**
- Events load for selected date
- Automatic refresh
- Shows all calendars you've granted access to

**Event Details:**
- Title displayed
- Time range shown
- Location included
- Calendar color preserved

**Opening Events:**
- Tap event in Afternoon
- Opens in Calendar app
- Full event details available
- Can edit in Calendar app

### Using with Time Blocking

**Planning Strategy:**
1. Check Calendar Only view
2. See existing commitments
3. Switch to Day Planner
4. Create time blocks around meetings
5. Schedule tasks in free time

**Benefits:**
- See full schedule in one place
- Plan realistically
- Avoid double-booking
- Protect focus time

### Privacy & Permissions

**What Afternoon Accesses:**
- Event titles
- Event times
- Event locations
- Calendar names

**What Afternoon Doesn't Access:**
- Event attendees
- Event notes
- Private event details
- Other calendar data

**Your Control:**
- Grant/revoke access anytime
- Choose which calendars to show
- Manage in iOS Settings

### Troubleshooting

**Events Not Showing:**
1. Check calendar permissions
2. Verify events exist in Calendar app
3. Ensure correct date selected
4. Switch to Day Planner or Calendar Only view

**Wrong Events Showing:**
1. Check which calendars are enabled
2. Manage in iOS Settings → Afternoon → Calendars
3. Disable unwanted calendars

**Events Not Opening:**
1. Ensure Calendar app is installed
2. Check iOS permissions
3. Try tapping event again

## Integration Best Practices

### Apple Health

1. **Choose Relevant Habits:**
   - Only sync health-related habits
   - Don't sync everything
   - Keep Health data meaningful

2. **Use Appropriate Units:**
   - Match units to Health categories
   - Minutes for meditation
   - Glasses for water
   - Sessions for workouts

3. **Be Consistent:**
   - Track daily for accurate Health data
   - Don't skip days unnecessarily
   - Maintain data quality

4. **Review Health App:**
   - Check synced data periodically
   - Verify accuracy
   - Adjust habits if needed

### Apple Calendar

1. **Plan Around Events:**
   - Check calendar first thing
   - Create time blocks around meetings
   - Leave buffer time

2. **Use Day Planner:**
   - Best view for calendar integration
   - See everything in timeline
   - Visual schedule planning

3. **Don't Duplicate:**
   - Don't create tasks for calendar events
   - Use calendar events as reference
   - Create time blocks if needed

4. **Regular Review:**
   - Check calendar daily
   - Plan week ahead
   - Adjust schedule as needed
